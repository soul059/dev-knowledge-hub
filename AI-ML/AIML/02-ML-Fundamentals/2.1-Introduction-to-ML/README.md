# 🤖 Introduction to Machine Learning

## Course Overview

This module covers the core concepts and algorithms of supervised machine learning. From understanding the ML workflow to implementing key algorithms like Linear Regression, Decision Trees, SVMs, and Ensemble Methods, these notes provide both theoretical foundations and practical implementation skills.

After completing this module, you will be able to:
- Understand the ML workflow from data to deployment

- Implement and evaluate supervised learning algorithms
- Choose the right algorithm for different problem types
- Tune hyperparameters for optimal performance

## 📊 Algorithm Comparison Table

| Algorithm | Type | Interpretable | Handles Nonlinear | Feature Scaling | Training Speed |
|-----------|------|:---:|:---:|:---:|:---:|
| Linear Regression | Regression | ✅ | ❌ (unless polynomial) | Recommended | Fast |
| Logistic Regression | Classification | ✅ | ❌ | Required | Fast |
| KNN | Both | ✅ | ✅ | Required | No training |
| Decision Trees | Both | ✅ | ✅ | Not needed | Fast |
| Random Forest | Both | ⚠️ | ✅ | Not needed | Medium |
| Gradient Boosting | Both | ⚠️ | ✅ | Not needed | Slow |
| SVM | Both | ⚠️ | ✅ (kernel) | Required | Slow (large data) |
| Naive Bayes | Classification | ✅ | ❌ | Not needed | Very Fast |
