## 1. Overview

Machine learning is broadly divided into three paradigms: **Supervised Learning (SL)**,
**Unsupervised Learning (UL)**, and **Reinforcement Learning (RL)**. Understanding how
they differ вҖ” in data, feedback, and objective вҖ” is essential for choosing the right
approach for a given problem.

## 2. Supervised Learning (SL)

### How It Works

The model learns a mapping $f: X \rightarrow Y$ from a dataset of input-output pairs
$\{(x_i, y_i)\}_{i=1}^{N}$.

$$\min_\theta \; \frac{1}{N} \sum_{i=1}^{N} \mathcal{L}\big(f_\theta(x_i),\; y_i\big)$$

where $\mathcal{L}$ is a loss function (e.g., MSE, cross-entropy).

### Characteristics

- **Data**: Labelled pairs $(x, y)$ provided upfront.
- **Feedback**: Immediate вҖ” the correct answer $y$ is known.
- **Goal**: Minimise prediction error on unseen data.
- **Examples**: Image classification, spam detection, language translation.

```python
# --- Supervised Learning Example (scikit-learn) ---
from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = LogisticRegression(max_iter=200)
model.fit(X_train, y_train)           # learns from labelled data
accuracy = model.score(X_test, y_test)
print(f"Test Accuracy: {accuracy:.2%}")
```

## 3. Unsupervised Learning (UL)

### How It Works

The model receives only inputs $\{x_i\}_{i=1}^{N}$ with **no labels**. It must discover
hidden structure, clusters, or representations.

$$\min_\theta \; \mathcal{L}(\theta; \; X)$$

Common objectives include reconstruction error (autoencoders), likelihood maximisation
(GMMs), or distance-based clustering (K-Means).

### Characteristics

- **Data**: Unlabelled $\{x_i\}$ only.
- **Feedback**: None вҖ” no correct answers exist.
- **Goal**: Discover structure, compress data, or detect anomalies.
- **Examples**: Customer segmentation, topic modelling, dimensionality reduction.

```python
# --- Unsupervised Learning Example (scikit-learn) ---
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs

X, _ = make_blobs(n_samples=300, centers=4, random_state=42)

kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
kmeans.fit(X)                          # no labels given
labels = kmeans.labels_
print(f"Cluster centres:\n{kmeans.cluster_centers_}")
print(f"Cluster sizes : {[sum(labels == i) for i in range(4)]}")
```

## 4. Reinforcement Learning (RL)

### How It Works

The agent interacts with an environment over a sequence of time steps, receiving a
scalar reward after each action. There is no dataset вҖ” data is generated **on-the-fly**
by the agent's own actions.

$$\max_\pi \; \mathbb{E}_\pi \left[ \sum_{t=0}^{T} \gamma^t \, r_{t+1} \right]$$

### Characteristics

- **Data**: Generated by the agent through interaction (not i.i.d.).
- **Feedback**: Delayed, scalar reward вҖ” not the correct action.
- **Goal**: Maximise cumulative reward over time.
- **Examples**: Game playing, robot control, dialogue systems.

```python
# --- Reinforcement Learning Example (Gymnasium) ---
import gymnasium as gym

env = gym.make("CartPole-v1")
state, info = env.reset(seed=42)

total_reward = 0
for step in range(200):
    action = env.action_space.sample()       # random policy (no labels!)
    state, reward, terminated, truncated, info = env.step(action)
    total_reward += reward
    if terminated or truncated:
        break

print(f"Episode lasted {step + 1} steps, total reward: {total_reward}")
env.close()
```

## 6. Comprehensive Comparison Table

| Dimension             | Supervised Learning       | Unsupervised Learning    | Reinforcement Learning      |
|-----------------------|---------------------------|--------------------------|-----------------------------|
| **Data**              | Labelled pairs $(x, y)$   | Unlabelled $\{x\}$      | Agent-generated $(s,a,r,s')$|
| **Feedback**          | Ground-truth labels       | None                     | Scalar reward (delayed)     |
| **Learning signal**   | Error / loss gradient     | Reconstruction / density | Reward signal / return      |
| **Goal**              | Predict $y$ from $x$      | Find structure in $x$    | Maximise cumulative reward  |
| **Data distribution** | i.i.d. assumed            | i.i.d. assumed           | Non-stationary, sequential  |
| **Evaluation**        | Accuracy, F1, MSE         | Silhouette, NMI          | Cumulative reward, regret   |
| **Exploration**       | Not needed                | Not needed               | Critical                    |
| **Credit assignment** | Direct (one step)         | N/A                      | Temporal (multi-step)       |
| **Common algorithms** | SVM, Random Forest, NNs   | K-Means, PCA, VAE       | Q-Learning, PPO, SAC        |
| **Example tasks**     | Image classification      | Clustering               | Game playing, robot control |

## 8. Key Differences in Code

To highlight the fundamental difference, here is the same "learn to pick the best
action" problem framed in all three paradigms:

```python
import numpy as np
np.random.seed(0)

# ===== SUPERVISED LEARNING =====
# We are told the correct action for each state
states_sl   = np.array([[1], [2], [3], [4], [5]])
labels_sl   = np.array([0, 1, 1, 0, 1])   # ground-truth actions

# Train a simple model (nearest-neighbour)
from sklearn.neighbors import KNeighborsClassifier
model_sl = KNeighborsClassifier(n_neighbors=1)
model_sl.fit(states_sl, labels_sl)
print("SL prediction for state [3]:", model_sl.predict([[3]]))


# ===== UNSUPERVISED LEARNING =====
# We only see states, no labels or rewards
from sklearn.cluster import KMeans
states_ul = np.array([[1], [2], [3], [8], [9], [10]])
kmeans = KMeans(n_clusters=2, random_state=0, n_init=10).fit(states_ul)
print("UL cluster labels:", kmeans.labels_)


# ===== REINFORCEMENT LEARNING =====
# No labels. We interact and get rewards.
n_actions = 2
Q = np.zeros(n_actions)
N = np.zeros(n_actions)

for _ in range(500):
    # epsilon-greedy exploration
    if np.random.rand() < 0.1:
        action = np.random.randint(n_actions)
    else:
        action = np.argmax(Q)

    # environment gives reward (unknown to agent a priori)
    reward = np.random.normal(loc=[0.5, 1.5][action], scale=0.5)
    N[action] += 1
    Q[action] += (reward - Q[action]) / N[action]

print(f"RL learned Q-values: {Q}  вҶ’  best action: {np.argmax(Q)}")
```

**Output:**
```
SL prediction for state [3]: [1]
UL cluster labels: [0 0 0 1 1 1]
RL learned Q-values: [0.47 1.52]  вҶ’  best action: 1
```

## 9. Mathematical Contrast

| Paradigm     | Objective Function                                                            |
|-------------|-------------------------------------------------------------------------------|
| Supervised  | $\min_\theta \frac{1}{N}\sum_{i=1}^{N} \mathcal{L}(f_\theta(x_i), y_i)$     |
| Unsupervised| $\min_\theta \sum_{i=1}^{N} \|x_i - \hat{x}_i(\theta)\|^2$ (reconstruction)  |
| RL          | $\max_\pi \mathbb{E}_\pi\big[\sum_{t=0}^{T}\gamma^t r_{t+1}\big]$            |

### Gradient Updates

- **SL**: $\theta \leftarrow \theta - \alpha \nabla_\theta \mathcal{L}$
- **UL**: $\theta \leftarrow \theta - \alpha \nabla_\theta \mathcal{L}_{recon}$
- **RL**: $\theta \leftarrow \theta + \alpha \nabla_\theta J(\pi_\theta)$ (policy gradient)

The RL update uses the **REINFORCE** estimator:

$$\nabla_\theta J(\pi_\theta) = \mathbb{E}_\pi \big[ \nabla_\theta \log \pi_\theta(a|s) \cdot G_t \big]$$

## 10. Hybrid Approaches

Modern ML often combines paradigms:

| Hybrid Method                  | Paradigms Combined | Example                     |
|-------------------------------|--------------------|-----------------------------|
| **RLHF**                      | SL + RL            | ChatGPT fine-tuning         |
| **Self-supervised + RL**       | UL + RL            | Pre-train representations   |
| **Imitation Learning**         | SL + RL            | Learn from expert demos     |
| **Semi-supervised Learning**   | SL + UL            | Limited labels + clustering |
| **Reward Modelling**           | SL вҶ’ RL            | Learn reward from preferences|
