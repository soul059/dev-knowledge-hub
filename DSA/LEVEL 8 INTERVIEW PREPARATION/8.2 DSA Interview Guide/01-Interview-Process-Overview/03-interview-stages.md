## Overview

The technical interview process is a **multi-stage pipeline**. Each stage has a different purpose, format, and elimination rate. Understanding the full pipeline helps you prepare for each stage strategically and manage your energy across the entire process.

---

## Stage 1: Application & Resume Screen

```
┌─────────────────────────────────────────────────────────┐
│  STAGE 1: APPLICATION                                    │
│                                                          │
│  ┌──────────────┐                                        │
│  │   RESUME     │──► Recruiter/ATS Screen                │
│  │              │                                        │
│  │ • Education  │    Decision Criteria:                   │
│  │ • Experience │    ├── Relevant experience?             │
│  │ • Skills     │    ├── Education background?            │
│  │ • Projects   │    ├── Keywords match?                  │
│  │ • Links      │    └── Referral available?              │
│  └──────────────┘                                        │
│                                                          │
│  Duration: 1-14 days                                     │
│  Your Action: Optimize resume, get referrals             │
│  Pass Rate: 10-20%                                       │
└─────────────────────────────────────────────────────────┘
```

### Tips for This Stage
- Tailor your resume to the job description
- Include relevant DSA projects or competitive programming achievements
- Get **referrals** — they can increase pass rate by 5-10x
- Include links to GitHub, LeetCode profile, or portfolio

## Stage 2: HR/Recruiter Phone Screen

```
┌─────────────────────────────────────────────────────────┐
│  STAGE 2: HR SCREEN                                      │
│                                                          │
│  Duration: 15-30 minutes                                 │
│  Format: Phone/Video call                                │
│                                                          │
│  Common Questions:                                       │
│  ┌────────────────────────────────────────────────┐      │
│  │ • Tell me about yourself                       │      │
│  │ • Why this company?                            │      │
│  │ • What's your timeline?                        │      │
│  │ • Salary expectations?                         │      │
│  │ • Visa/work authorization status?              │      │
│  │ • When can you start?                          │      │
│  └────────────────────────────────────────────────┘      │
│                                                          │
│  NOT technical — but still important!                    │
│  Pass Rate: ~50%                                         │
└─────────────────────────────────────────────────────────┘
```

### What They're Evaluating
- Communication skills and professionalism
- Genuine interest in the company and role
- Logistics compatibility (timeline, location, visa)
- Basic cultural alignment

## Stage 3: Technical Phone Screen

```
┌─────────────────────────────────────────────────────────┐
│  STAGE 3: TECHNICAL SCREEN                               │
│                                                          │
│  ┌──────────────────────────────────────────┐            │
│  │         45-60 MINUTE SESSION             │            │
│  │                                          │            │
│  │  0-5 min    │ Introduction & small talk   │            │
│  │  5-10 min   │ Problem statement given     │            │
│  │  10-15 min  │ Clarify & discuss approach  │            │

│  Platform: CoderPad, HackerRank, Google Docs             │

│  Pass Rate: 30-40%                                       │
└─────────────────────────────────────────────────────────┘
```

### Scoring Systems

| Rating | FAANG Term | Meaning |
|--------|-----------|---------|
| 5 | Strong Hire | Exceptional performance, exceeded expectations |
| 4 | Hire | Solid performance, met the bar |
| 3 | Lean Hire | Borderline, showed potential but had gaps |
| 2 | Lean No Hire | Below the bar, significant concerns |
| 1 | Strong No Hire | Clear rejection, major red flags |

## Quick Revision Questions

1. **What is the overall pass rate from application to offer at top companies?**
   - Approximately 1-3% of all applicants receive an offer

2. **How should you allocate time in a 45-minute technical screen?**
   - 25% time before coding (clarify + approach), 50% coding, 15% testing, 10% Q&A

3. **How many on-site rounds should you expect at a FAANG company?**
   - 5-6 rounds: typically 3 DSA, 1 System Design, 1 Behavioral, plus optional lunch

4. **What happens after the on-site interviews?**
   - Each interviewer submits independent written feedback; a hiring committee reviews all feedback together to make a decision

5. **Why should you always negotiate an offer?**
   - Companies expect negotiation; initial offers rarely represent the maximum they can pay; competing offers provide leverage

6. **What's the single best thing you can do to improve your application pass rate?**
   - Get an employee referral — it can increase pass rate by 5-10x compared to cold applications
