## Overview

Without clear goals, interview preparation becomes an endless, anxiety-driven loop of solving random problems. **Goal setting** transforms chaotic studying into a structured, measurable process with clear milestones.

---

## The SMART Goal Framework for DSA Prep

```
┌─────────────────────────────────────────────────────────┐
│              S.M.A.R.T. GOALS                           │
│                                                         │
│  S ─ Specific      "Solve 3 Medium DP problems/day"    │
│                     NOT "Get better at DP"              │
│                                                         │
│  M ─ Measurable    "Complete 80 Medium problems"        │
│                     NOT "Do a lot of problems"          │
│                                                         │
│  A ─ Achievable    "4 hours/day for 12 weeks"           │
│                     NOT "10 hours/day for 2 weeks"      │
│                                                         │
│  R ─ Relevant      "Focus on Arrays & Trees (most      │
│                      asked)" NOT "Study red-black trees"│
│                                                         │
│  T ─ Time-bound    "Ready by March 15th"                │
│                     NOT "Someday when I feel ready"     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## Goal Adjustment Strategy

```
IF progress is AHEAD of schedule:
┌──────────────────────────────────┐
│ • Add more Hard problems         │
│ • Increase mock interview count  │
│ • Start company-specific prep    │
│ • Help others (teaching solidifies learning) │
└──────────────────────────────────┘

IF progress is BEHIND schedule:
┌──────────────────────────────────┐
│ • Focus on high-frequency topics │
│ • Reduce Hard problems, increase Medium │
│ • Prioritize patterns over individual problems │
│ • Add 30 min/day or reduce rest days │
│ • DON'T panic — adjust, don't abandon │
└──────────────────────────────────┘

IF feeling BURNED OUT:
┌──────────────────────────────────┐
│ • Take 2-3 days off completely   │
│ • Reduce daily hours by 30%      │
│ • Switch to easier/fun problems  │
│ • Do social activities           │
│ • Remember: rest is part of prep │
└──────────────────────────────────┘
```

## Quick Revision Questions

1. **Why should you focus on process goals over outcome goals?**
   - Process goals are fully within your control; outcome goals depend on external factors like interviewer mood, competition, and luck

2. **What does the SMART acronym stand for in goal setting?**
   - Specific, Measurable, Achievable, Relevant, Time-bound

3. **What are the four topic mastery levels?**
   - Awareness, Understanding, Application, Mastery — aim for Level 3-4 on must-know topics

4. **When should you hit the "Interview Ready" checkpoint?**
   - By Week 9, you should solve 80%+ Medium problems in 25 min and score 3.5+/5 on mocks

5. **What should you do if you're behind schedule?**
   - Focus on high-frequency topics, prioritize patterns over individual problems, and add small increments of time — don't panic

6. **How often should you review and adjust your goals?**
   - Weekly — use the reflection template to assess what worked, what didn't, and adjust the next week's plan
