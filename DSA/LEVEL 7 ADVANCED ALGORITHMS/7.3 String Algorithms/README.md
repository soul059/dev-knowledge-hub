## ðŸ—ºï¸ Course Overview

Strings are among the most frequently encountered data types in programming
contests, system design, and real-world software. This module builds a deep,
intuition-first understanding of the algorithms that power search engines,
bioinformatics tools, compilers, and plagiarism detectors.

### What You Will Learn

| Skill | Description |
|---|---|
| **Pattern Matching** | KMP, Rabin-Karp, Z Algorithm |
| **Hashing on Strings** | Polynomial hash, rolling hash, double hashing |
| **Trie-Based Techniques** | Autocomplete, spell-check, XOR queries |
| **Palindrome Algorithms** | Manacher's algorithm |
| **Dynamic Programming on Strings** | LCS, edit distance, regex matching |
| **Advanced Structures** | Suffix arrays, suffix trees, Aho-Corasick |

### Prerequisites

- Arrays, Linked Lists, Stacks, Queues
- Recursion & Backtracking
- Basic Dynamic Programming
- Trie data structure (basic understanding)
