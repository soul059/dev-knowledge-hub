## Course Overview

This module covers **Recursion and Backtracking** â€” two of the most fundamental algorithmic paradigms in computer science. Recursion is the foundation for divide-and-conquer, dynamic programming, and tree/graph algorithms. Backtracking extends recursion to systematically explore solution spaces and is essential for constraint satisfaction problems.

**What you'll learn:**
- How to think recursively and break problems into subproblems
- Visualize the call stack and recursion trees
- Master backtracking patterns for combinatorial problems
- Solve classic problems: N-Queens, Sudoku, subsets, permutations
- Optimize with pruning, memoization, and symmetry breaking

## Complexity Quick Reference

| Pattern | Time Complexity | Space Complexity |
|---------|----------------|-----------------|
| Linear recursion | O(n) | O(n) |
| Binary recursion (Fibonacci) | O(2^n) | O(n) |
| Subsets/Subsequences | O(2^n) | O(n) |
| Permutations | O(n!) | O(n) |
| Binary search (recursive) | O(log n) | O(log n) |
| Divide and conquer (merge sort) | O(n log n) | O(n) |
