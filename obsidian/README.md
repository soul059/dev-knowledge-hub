## 🎯 Overview

Obsidian is a powerful knowledge base that works on local Markdown files. This folder contains foundational resources for using Obsidian effectively.

## 💡 Key Concepts

- **Markdown-Based**: All notes are stored as plain Markdown files
- **Local-First**: Your data stays on your device
- **Bi-directional Links**: Connect notes with `[[wiki-style links]]`
- **Graph View**: Visualize connections between notes
- **Plugins**: Extend functionality with community plugins

## 🛠️ Getting Started

1. Download Obsidian from [obsidian.md](https://obsidian.md)
2. Create or open a vault (folder for your notes)
3. Start writing in Markdown
4. Link notes together using `[[note name]]`
