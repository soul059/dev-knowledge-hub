# Unit 2: CompTIA Security+ — Topic 1: Exam Objectives Overview

## Overview

The **CompTIA Security+ (SY0-701)** is the industry-standard entry-level cybersecurity certification. This topic provides a comprehensive overview of the exam structure, domains, question types, and preparation strategy.

---

## 1. Exam Details

```
SY0-701 EXAM SPECIFICATIONS:

  Exam Code:       SY0-701
  Launch Date:     November 2023
  Questions:       Up to 90
  Question Types:  Multiple choice + Performance-based (PBQ)
  Duration:        90 minutes
  Passing Score:   750 out of 900
  Cost:            $404 USD
  Testing Center:  Pearson VUE (in-person or online)
  Prerequisites:   None (Network+ recommended)
  Valid:           3 years (then renew via CEUs)

QUESTION BREAKDOWN:
  → Multiple choice: ~75% of questions
  → Performance-based: ~25% of questions
  → PBQs test practical skills:
    - Drag and drop configurations
    - Network diagram analysis
    - Command line operations
    - Matching exercises
    - Simulation scenarios

SCORING:
  → Scale: 100-900
  → Pass: 750/900
  → Not a percentage — scaled scoring
  → PBQs may be weighted differently
  → No penalty for guessing
  → Review flagged questions before submitting
```

## 4. Exam Strategy

```
EXAM PREPARATION STRATEGY:

12-WEEK STUDY PLAN:
  Week 1-2:  Domain 1 (General Concepts)
  Week 3-5:  Domain 2 (Threats & Vulns)
  Week 5-7:  Domain 3 (Architecture)
  Week 7-10: Domain 4 (Operations)
  Week 10-12: Domain 5 (Program Mgmt)
  Week 12-14: Review + Practice Exams

EXAM DAY STRATEGY:
  1. Skip PBQs initially — do MCQs first
  2. Flag uncertain questions for review
  3. Manage time (1 min per MCQ)
  4. Return to PBQs with remaining time
  5. Review flagged questions
  6. Never leave blanks — guess if unsure
  7. Read questions carefully — watch for NOT/EXCEPT
  8. Eliminate obviously wrong answers first
  9. Look for absolute words (always, never) = usually wrong
  10. "Best" answer vs correct answer

PRACTICE EXAM SCORING TARGET:
  Score on Practice | Ready?
  < 70%             | Not yet — study more
  70-80%            | Getting close — focus weak areas
  80-85%            | Nearly ready — targeted review
  85%+              | Ready — schedule exam
```
