# Unit 1: Certification Landscape — Topic 2: Mid-Level Certifications

## Overview

**Mid-level certifications** validate practical skills and deeper knowledge for security professionals with 2-5 years of experience. These certifications demonstrate competency in specific domains like penetration testing, incident response, or security engineering and are often required for senior analyst or engineer positions.

---

## 1. CEH (Certified Ethical Hacker)

```
CEH v12:

  DETAILS:
  Vendor:        EC-Council
  Exam Code:     312-50v12
  Questions:     125 multiple choice
  Duration:      4 hours
  Passing Score: ~70% (varies)
  Cost:          $1,199 (exam only)
  Prerequisites: 2 years InfoSec experience OR
                 official training
  Renewal:       Every 3 years (ECE credits)

  DOMAINS:
  → Information security & ethical hacking
  → Reconnaissance techniques
  → Scanning networks
  → Enumeration
  → Vulnerability analysis
  → System hacking
  → Malware threats
  → Sniffing
  → Social engineering
  → Denial-of-service
  → Session hijacking
  → Hacking web servers & applications
  → SQL injection
  → Wireless network hacking
  → Mobile platform hacking
  → IoT and OT hacking
  → Cloud computing
  → Cryptography

  PROS:
  → Widely recognized
  → DoD 8570 approved
  → Broad ethical hacking coverage
  → Good for job requirements
  → Global recognition

  CONS:
  → Expensive with training
  → Theory-heavy (multiple choice)
  → Practical exam (CEH Practical) separate
  → Some consider outdated approach
  → Better practical alternatives exist
```

## 2. CompTIA PenTest+

```
CompTIA PenTest+ (PT0-002):

  DETAILS:
  Vendor:        CompTIA
  Exam Code:     PT0-002
  Questions:     85 (multiple choice + PBQ)
  Duration:      165 minutes
  Passing Score: 750/900
  Cost:          ~$404 USD
  Prerequisites: None (Security+ recommended)
  Experience:    3-4 years recommended

  DOMAINS:
  Domain                          | Weight
  1. Planning and Scoping         | 14%
  2. Information Gathering        | 22%
  3. Attacks and Exploits         | 30%
  4. Reporting and Communication  | 18%
  5. Tools and Code Analysis      | 16%

  PROS:
  → More affordable than CEH
  → Hands-on performance questions
  → DoD 8570 approved
  → Good stepping stone to OSCP
  → Vendor-neutral
  → Includes reporting skills

  IDEAL PATH:
  Security+ → PenTest+ → OSCP
```

## 3. SSCP (Systems Security Certified Practitioner)

```
(ISC)² SSCP:

  DETAILS:
  Vendor:        (ISC)²
  Questions:     125 multiple choice
  Duration:      3 hours
  Passing Score: 700/1000
  Cost:          ~$249 USD
  Prerequisites: 1 year experience in domain
  Renewal:       Every 3 years (CPE credits)

  DOMAINS:
  Domain                          | Weight
  1. Security Operations          | 16%
  2. Access Controls              | 15%
  3. Risk Identification          | 15%
  4. Incident Response            | 13%
  5. Cryptography                 | 9%
  6. Network & Communication      | 16%
  7. Systems & Application        | 16%

  IDEAL FOR:
  → Hands-on security administrators
  → Stepping stone to CISSP
  → Network security engineers
  → System administrators with security focus

  vs CISSP:
  → SSCP = technical practitioner
  → CISSP = management/governance
  → SSCP requires 1 year vs CISSP's 5 years
```

## 4. Other Mid-Level Certifications

```
ADDITIONAL MID-LEVEL CERTS:

CASP+ (CompTIA Advanced Security Practitioner):
  → Advanced technical certification
  → No multiple choice — performance-based
  → Enterprise security focus
  → DoD 8570 IAT Level III
  → ~$494 USD
  → Bridge between mid and advanced

Cisco CyberOps Associate:
  → SOC analyst focus
  → Cisco technology aligned
  → Network security monitoring
  → Good for Cisco environments

CISM (Certified Information Security Manager):
  → ISACA certification
  → Security management focus
  → 5 years experience required
  → Business-focused security
  → Good for management track

CCNA Security (now part of CCNA):
  → Network security focus
  → Cisco-specific
  → Good for network security roles

eJPT (eLearnSecurity Junior Penetration Tester):
  → INE/eLearnSecurity
  → Practical exam (black-box pentest)
  → Good OSCP stepping stone
  → Affordable (~$249)
  → 100% hands-on

eCPPTv2:
  → eLearnSecurity Certified Professional PT
  → Practical exam with report
  → Good OSCP preparation
  → More approachable than OSCP
```

## 5. Mid-Level Comparison and Paths

```
MID-LEVEL COMPARISON:

  Cert      | Cost   | Type      | Focus       | Difficulty
  CEH       | $1,200 | MCQ       | Ethical hack| Medium
  PenTest+  | $404   | MCQ + PBQ | Pen testing | Medium
  SSCP      | $249   | MCQ       | Security ops| Medium
  CASP+     | $494   | PBQ       | Advanced    | Medium-Hard
  eJPT      | $249   | Practical | Pen testing | Easy-Med
  eCPPTv2   | $400   | Practical | Pen testing | Medium
  CySA+     | $404   | MCQ + PBQ | Blue team   | Medium

CAREER PATH ALIGNMENT:
  Red Team Path:
  Security+ → eJPT → PenTest+ → eCPPTv2 → OSCP

  Blue Team Path:
  Security+ → CySA+ → SSCP → GCIH → CISSP

  Management Path:
  Security+ → SSCP → CISM → CISSP

  Network Security:
  CCNA → CyberOps → Security+ → CASP+

WHEN TO GET MID-LEVEL CERTS:
  → 2-4 years into security career
  → After establishing foundational knowledge
  → When job requirements demand it
  → To specialize in a domain
  → Before attempting advanced certs
```
