# Unit 1: Certification Landscape — Topic 4: Specialist Certifications

## Overview

**Specialist certifications** focus on deep expertise in narrow domains — cloud security, forensics, malware analysis, application security, or specific vendor technologies. These certifications demonstrate mastery in a particular area and are valuable for roles requiring specialized skills.

---

## 1. Cloud Security Specializations

```
CLOUD SECURITY CERTS:

AWS Certified Security - Specialty:
  → AWS-specific security services
  → IAM, KMS, CloudTrail, GuardDuty
  → Requires AWS experience
  → ~$300 USD
  → Valid 3 years

Microsoft AZ-500 (Azure Security Engineer):
  → Azure security services
  → Sentinel, Defender, Key Vault
  → Identity protection
  → ~$165 USD
  → Valid 1 year (renewal free)

Microsoft SC-200 (Security Operations Analyst):
  → Microsoft Sentinel
  → Microsoft 365 Defender
  → Defender for Cloud
  → SOC analyst focus

Google Professional Cloud Security Engineer:
  → GCP security controls
  → VPC, IAM, Cloud Armor
  → ~$200 USD

CCSP ((ISC)² Cloud Security):
  → Vendor-neutral cloud security
  → 5 years experience
  → Comprehensive cloud domains
  → ~$599 USD

CCSK (Certificate of Cloud Security Knowledge):
  → CSA (Cloud Security Alliance)
  → Entry point for cloud security
  → Open-book exam
  → ~$395 USD
```

## 2. Offensive Security Specializations

```
OFFENSIVE SECURITY CERTS:

OffSec (Offensive Security):
  OSCP: Penetration testing (covered in Unit 4)
  OSWE: Web application exploitation
  OSEP: Evasion techniques and breaching
  OSED: Exploit development
  OSDA: Defense analyst
  OSMR: macOS researcher

  Certification Path:
  OSCP → OSWE (web) → OSEP (evasion) → OSED (exploit)

  OSWE Details:
  → WEB-300 course
  → 48-hour practical exam
  → White-box web app attacks
  → Custom exploit development
  → Code review skills

  OSEP Details:
  → PEN-300 course
  → 48-hour practical exam
  → AV evasion
  → Advanced lateral movement
  → Phishing for access
  → Custom C2 usage

SANS/GIAC Offensive:
  GPEN: Penetration Tester
  GWAPT: Web Application Penetration Tester
  GXPN: Exploit Researcher and Advanced Pen Tester
  GMOB: Mobile Device Security Analyst

Other Offensive:
  CRTP: Certified Red Team Professional (Pentester Academy)
  → Active Directory focused
  → Practical exam
  → Affordable (~$300)

  CRTO: Certified Red Team Operator (Zero Point Security)
  → Cobalt Strike focused
  → Red team operations
  → ~$450

  PNPT: Practical Network Penetration Tester (TCM)
  → Affordable practical cert
  → Includes OSINT, report writing
  → ~$400
```

## 3. Blue Team Specializations

```
DEFENSIVE SECURITY CERTS:

SANS/GIAC Defensive:
  GCIH: Incident Handler
  → Incident response methodology
  → Attack/intrusion techniques
  → Highly respected

  GCIA: Intrusion Analyst
  → Network traffic analysis
  → Intrusion detection
  → Deep packet analysis

  GCFE: Forensic Examiner
  → Windows forensics
  → File system analysis
  → Registry forensics

  GCFA: Forensic Analyst
  → Advanced forensics
  → Timeline analysis
  → Memory forensics

  GNFA: Network Forensic Analyst
  → Network forensics
  → Traffic analysis
  → Evidence collection

Threat Intelligence:
  GCTI: Cyber Threat Intelligence (GIAC)
  → Intelligence analysis
  → ATT&CK mapping
  → Reporting

  CTIA: Certified Threat Intelligence Analyst (EC-Council)
  → Threat intel lifecycle
  → IOC analysis
  → Reporting

SOC/IR:
  CySA+: CompTIA (covered in mid-level)
  ECIH: EC-Council Incident Handler
  BTL1: Blue Team Level 1 (Security Blue Team)
  → Practical exam
  → Affordable
  → Good for SOC careers
  BTL2: Blue Team Level 2
  → Advanced practical
  → Incident response focus
```

## 4. Forensics and Malware Specializations

```
FORENSICS CERTS:

GIAC Forensics Path:
  GCFE → GCFA → GNFA
  → Windows → Advanced → Network
  → Highly technical
  → Very expensive with SANS training

EnCase Certified Examiner (EnCE):
  → EnCase tool certification
  → Digital forensics
  → Court-accepted
  → Requires EnCase experience

AccessData Certified Examiner (ACE):
  → FTK tool certification
  → Forensic toolkit proficiency

CFCE (Certified Forensic Computer Examiner):
  → IACIS certification
  → Law enforcement focused
  → Peer review process

CCE (Certified Computer Examiner):
  → ISFCE certification
  → Vendor-neutral
  → Practical + written exam

MALWARE ANALYSIS CERTS:

GREM (GIAC Reverse Engineering Malware):
  → Malware analysis methodology
  → Reverse engineering
  → Behavioral analysis
  → Highly technical

CREA (Certified Reverse Engineering Analyst):
  → IACRB certification
  → Advanced reverse engineering

eCMAP (eLearnSecurity Malware Analysis Professional):
  → Practical exam
  → INE/eLearnSecurity
  → Hands-on analysis
```

## 5. Vendor-Specific Specializations

```
VENDOR-SPECIFIC CERTS:

CISCO:
  → CCNP Security
  → CyberOps Professional
  → Cisco Certified Specialist (various)

PALO ALTO:
  → PCNSA (Network Security Administrator)
  → PCNSE (Network Security Engineer)
  → PCSAE (Security Automation Engineer)
  → PCCSE (Cloud Security Engineer)

FORTINET:
  → NSE 4-8 (Network Security Expert)
  → Free training available
  → Good for Fortinet shops

SPLUNK:
  → Splunk Core Certified User
  → Splunk Core Certified Power User
  → Splunk Certified Admin
  → Splunk Enterprise Security Certified Admin

CHECK POINT:
  → CCSA (Security Administrator)
  → CCSE (Security Expert)
  → CCSM (Security Master)

CROWDSTRIKE:
  → CCFR (Falcon Responder)
  → CCFA (Falcon Administrator)
  → CCFH (Falcon Hunter)

WHEN TO GET VENDOR CERTS:
  → When your organization uses the vendor
  → For roles requiring specific tool expertise
  → To complement vendor-neutral certs
  → For career advancement within an ecosystem
  → Vendor certs + CISSP = strong combination
```
