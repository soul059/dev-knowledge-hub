# Unit 1: Certification Landscape — Topic 1: Entry-Level Certifications

## Overview

**Entry-level certifications** establish foundational knowledge in cybersecurity and are designed for individuals beginning their security career. These certifications validate understanding of core concepts, terminology, and basic security practices without requiring extensive prior experience.

---

## 1. CompTIA Security+

```
CompTIA Security+ (SY0-701):

  STATUS: Industry gold standard for entry-level

  DETAILS:
  Vendor:        CompTIA
  Exam Code:     SY0-701 (current)
  Questions:     90 (multiple choice + PBQ)
  Duration:      90 minutes
  Passing Score: 750/900
  Cost:          ~$404 USD
  Prerequisites: None (Network+ recommended)
  Renewal:       Every 3 years (CEUs)
  Experience:    0-2 years recommended

  DOMAINS:
  Domain                          | Weight
  1. General Security Concepts    | 12%
  2. Threats, Vulns & Mitigations | 22%
  3. Security Architecture        | 18%
  4. Security Operations          | 28%
  5. Security Program Mgmt        | 20%

  WHY GET IT:
  → DoD 8570 / 8140 baseline requirement
  → Recognized globally
  → Required by many job postings
  → Vendor-neutral
  → Gateway to more advanced certs
  → Career starter in security

  CAREERS IT SUPPORTS:
  → Security Analyst (entry)
  → SOC Analyst
  → IT Security Specialist
  → Systems Administrator (security)
  → Help Desk (security-focused)
```

## 2. CompTIA CySA+

```
CompTIA CySA+ (CS0-003):

  STATUS: Defensive security analyst certification

  DETAILS:
  Vendor:        CompTIA
  Exam Code:     CS0-003 (current)
  Questions:     85 (multiple choice + PBQ)
  Duration:      165 minutes
  Passing Score: 750/900
  Cost:          ~$404 USD
  Prerequisites: None (Security+ recommended)
  Experience:    3-4 years recommended

  DOMAINS:
  Domain                          | Weight
  1. Security Operations          | 33%
  2. Vulnerability Management     | 30%
  3. Incident Response            | 20%
  4. Reporting & Communication    | 17%

  WHY GET IT:
  → Bridges gap between Security+ and advanced
  → Focused on blue team / defensive skills
  → DoD 8570 approved
  → Hands-on performance questions
  → Good for SOC analyst career path

  IDEAL FOR:
  → SOC analysts moving to senior roles
  → Threat analysts
  → Vulnerability management analysts
  → Security engineers (entry)
```

## 3. Other Entry-Level Certifications

```
ADDITIONAL ENTRY-LEVEL CERTS:

CompTIA Network+:
  → Networking fundamentals
  → Prerequisite knowledge for Security+
  → Good starting point
  → ~$358 USD

CompTIA A+:
  → IT fundamentals
  → Hardware/software basics
  → Not security-specific
  → Starting point for IT career

(ISC)² CC (Certified in Cybersecurity):
  → FREE certification and training
  → Entry-level security concepts
  → Good stepping stone to SSCP/CISSP
  → No experience required

ISACA CSX-P:
  → Cybersecurity Practitioner
  → Performance-based
  → Lab-based exam

EC-Council CND (Certified Network Defender):
  → Network defense focus
  → Entry to mid-level
  → Good for network security roles

Microsoft SC-900:
  → Security, Compliance, Identity Fundamentals
  → Free training available
  → Good for Microsoft-focused roles
  → ~$165 USD

AWS Cloud Practitioner + Security:
  → Cloud security basics
  → Good for cloud career path
  → AWS-focused environment

Google Cybersecurity Certificate:
  → Coursera-based
  → Entry-level
  → Self-paced
  → ~$300 total
```

## 5. Preparing for Entry-Level Certs

```
STUDY APPROACH:

TIMELINE (Security+):
  Week 1-2:  Domain 1 - General Concepts
  Week 3-5:  Domain 2 - Threats & Vulnerabilities
  Week 5-7:  Domain 3 - Security Architecture
  Week 7-10: Domain 4 - Security Operations
  Week 10-12: Domain 5 - Program Management
  Week 12-14: Review and practice exams

STUDY RESOURCES:
  → Professor Messer (free YouTube)
  → CompTIA CertMaster
  → Darril Gibson study guide
  → Mike Meyers Udemy course
  → Practice exams (Dion Training, ExamCompass)

STUDY TIPS:
  [ ] Set a firm exam date
  [ ] Study 1-2 hours daily
  [ ] Take notes by hand
  [ ] Use flashcards for terms
  [ ] Practice with hands-on labs
  [ ] Take multiple practice exams
  [ ] Score 85%+ on practice before real exam
  [ ] Review wrong answers thoroughly
  [ ] Join study groups (Reddit, Discord)
  [ ] Use PBQ practice (performance-based)
```
