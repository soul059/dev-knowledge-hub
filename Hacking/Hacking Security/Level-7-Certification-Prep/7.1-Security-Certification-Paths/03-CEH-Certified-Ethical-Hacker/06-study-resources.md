# Unit 3: CEH — Topic 6: Study Resources and Exam Preparation

## Overview

This topic consolidates the best resources, study strategies, and exam-day tips for passing the CEH v12 (312-50) certification exam. It covers official and third-party materials, lab platforms, and preparation timelines.

---

## 4. Study Plan and Timeline

```
16-WEEK STUDY PLAN:

  Week  | Modules          | Focus
  1-2   | Intro + Recon    | Footprinting, scanning
  3-4   | Enumeration      | Services, protocols
  5-6   | Vulnerability    | Scanning, assessment
  7-8   | System Hacking   | Passwords, escalation
  9-10  | Malware + SE     | Trojans, phishing
  11-12 | Web + Wireless   | OWASP, WiFi cracking
  13-14 | Cloud + Mobile   | IoT, containers
  15    | Crypto + Review  | PKI, hashing
  16    | Practice exams   | Test readiness

DAILY ROUTINE (2-3 hours):
  → 45 min: Study material (read/watch)
  → 45 min: Hands-on labs
  → 30 min: Practice questions (30 Qs)
  → 15 min: Review incorrect answers
  → 15 min: Tool practice

STUDY TECHNIQUES:
  → Create mind maps per module
  → Build flashcards for tools
  → Practice in labs daily
  → Join study groups (Discord, Reddit)
  → Teach concepts to others
  → Track progress per domain

READINESS CHECKLIST:
  [ ] Score 80%+ on 3 different practice exams
  [ ] Completed labs for all modules
  [ ] Can identify all tools by category
  [ ] Know Google dorking operators
  [ ] Can explain all attack phases
  [ ] Memorized key port numbers
  [ ] Know cryptographic algorithms
  [ ] Can read Nmap/Wireshark output
  [ ] Understand all wireless attacks
  [ ] Know OWASP Top 10
```

## 5. Exam Day and Beyond

```
EXAM DAY STRATEGY:

BEFORE EXAM:
  → Good night's sleep
  → Eat well, stay hydrated
  → Arrive early with valid ID
  → Review quick reference notes
  → Don't cram new material

DURING EXAM (125 Qs, 4 hours):
  → ~1.9 minutes per question
  → Read questions completely
  → Eliminate obviously wrong answers
  → Watch for "BEST" and "MOST" qualifiers
  → Flag uncertain questions
  → Don't change answers unless sure
  → Manage time (check at Q30, Q60, Q90)

QUESTION TYPES:
  → Tool identification scenarios
  → Attack methodology order
  → Output interpretation
  → "Which tool would you use?"
  → Legal/ethical scenarios
  → Countermeasure selection

AFTER PASSING:
  → Download certificate
  → Add to resume/LinkedIn
  → Earn ECE credits (120 in 3 years)
  → Consider CEH Practical
  → Plan next certification:
    → OSCP (practical offensive)
    → CySA+ (defensive analysis)
    → CHFI (forensics)
    → CPENT (advanced pentesting)

CAREER OPPORTUNITIES:
  Position              | Salary Range
  Ethical Hacker        | $75,000-$120,000
  Penetration Tester    | $80,000-$130,000
  Security Analyst      | $65,000-$100,000
  Security Consultant   | $85,000-$140,000
  SOC Analyst           | $55,000-$90,000
```
