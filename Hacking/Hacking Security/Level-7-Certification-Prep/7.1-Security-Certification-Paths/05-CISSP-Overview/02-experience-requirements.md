# Unit 5: CISSP Overview — Topic 2: Experience Requirements

## Overview

The CISSP has the most stringent experience requirements of any major security certification. Understanding the exact requirements, acceptable experience, and alternative pathways is essential for planning your certification journey.

---

## 1. Standard Experience Requirements

```
CISSP EXPERIENCE REQUIREMENTS:

STANDARD PATH:
  → 5 years cumulative paid work experience
  → Must be in 2 or more of the 8 domains
  → Full-time experience required
  → Part-time: 1,040 hours = 6 months
  → Experience can be accumulated over time

WAIVER OPTIONS:
  4-Year Path (with education/credential):
  → 4-year college degree (any field) = 1 year waiver
  → Master's degree in IS/IT = 1 year waiver
  → Approved (ISC)² credentials = 1 year waiver
  → Maximum waiver: 1 year only

  Approved Credentials for Waiver:
  → CCNA, CCNP, CCIE
  → CEH, CHFI
  → CompTIA Security+, CySA+, CASP+
  → MCSE, MCITP
  → GSEC, GCIA, GCIH
  → SSCP
  → Many others on (ISC)² list

ASSOCIATE OF (ISC)²:
  → Pass CISSP exam WITHOUT experience
  → Receive "Associate" designation
  → 6 years to earn required experience
  → Then endorsement for full CISSP
  → Good option for career changers

EXPERIENCE DOCUMENTATION:
  → Detailed resume with dates
  → Job descriptions mapping to domains
  → Supervisor information
  → Subject to (ISC)² audit
  → Random audits post-endorsement
```

## 3. Endorsement Process

```
ENDORSEMENT STEPS:

AFTER PASSING THE EXAM:
  1. Receive pass notification
  2. Submit endorsement application
  3. Complete online endorsement form
  4. Provide experience details
  5. Obtain endorsing (ISC)² member
  6. Submit for (ISC)² review

ENDORSEMENT REQUIREMENTS:
  → Must have endorser who is active (ISC)²
    certified professional
  → Endorser attests to your experience
  → Endorser does NOT need to have
    worked with you directly
  → Can find endorser through:
    - Professional network
    - Local (ISC)² chapter
    - (ISC)² online community
    - LinkedIn connections
    - (ISC)² will act as endorser if needed

TIMELINE:
  → Submit within 9 months of passing
  → (ISC)² reviews in 4-8 weeks
  → May request additional documentation
  → Random audits occur
  → If audited: provide detailed proof

AUDIT PROCESS:
  → Random selection after endorsement
  → Must provide:
    - Detailed job descriptions
    - Dates of employment
    - Supervisor contact information
    - Mapping of duties to domains
  → Failure to comply = certification denied
  → Be honest — fraud = permanent ban
```

## 4. Career Planning for CISSP

```
CAREER PATH TO CISSP:

TYPICAL TIMELINE:

  Year 0:   Start IT career
  Year 1-2: Help desk, IT support
             → Get CompTIA Security+
  Year 2-3: Junior security role
             → SOC Analyst, Security Admin
  Year 3-4: Mid-level security role
             → Security Analyst, Engineer
  Year 4-5: Senior security role
             → Study for CISSP
  Year 5+:  Management/architect role
             → Earn CISSP

ROLES THAT BUILD CISSP EXPERIENCE:
  Entry Level (Years 1-2):
  → IT Support Specialist
  → Junior Systems Administrator
  → Network Technician
  → Help Desk Analyst

  Mid Level (Years 2-4):
  → Security Analyst
  → SOC Analyst (Tier 1-2)
  → Systems Administrator
  → Network Administrator
  → IT Auditor

  Senior Level (Years 4-5+):
  → Senior Security Analyst
  → Security Engineer
  → Security Architect
  → IT Security Manager
  → Incident Response Lead

POST-CISSP ROLES:
  → Chief Information Security Officer (CISO)
  → Security Director
  → Security Architect
  → Security Consultant
  → Risk Manager
  → Compliance Officer
  → Salary: $100,000-$200,000+
```

## 5. Alternative Certifications While Building Experience

```
STEPPING STONE CERTIFICATIONS:

IF YOU DON'T HAVE ENOUGH EXPERIENCE:

  Certification    | Experience Req | Leads To
  CompTIA Security+| None           | Entry roles
  SSCP             | 1 year         | CISSP path
  CySA+            | None           | SOC roles
  CISM             | 5 years        | Management
  CISA             | 5 years        | Audit
  CCSP             | 5 years        | Cloud sec

RECOMMENDED PATH:
  1. CompTIA Security+ (0 experience)
  2. CySA+ or CEH (1-2 years)
  3. SSCP (1 year experience)
     → (ISC)² family, good CISSP prep
  4. CISSP (5 years experience)

SSCP (Systems Security Certified Practitioner):
  → (ISC)² certification
  → 1 year experience required
  → 7 domains (subset of CISSP)
  → 125 questions, 3 hours
  → Good stepping stone to CISSP
  → Counts toward CISSP experience waiver

ASSOCIATE PATH:
  → Pass CISSP exam now
  → Earn "Associate of (ISC)²"
  → Build experience over 6 years
  → Convert to full CISSP when ready
  → Valid option for career changers
  → Shows commitment to certification

MAINTAINING ASSOCIATE STATUS:
  → 15 CPE credits per year
  → Annual maintenance fee ($50)
  → 6-year window to earn experience
  → Must submit endorsement when ready
```
