# Unit 5: CISSP Overview — Topic 3: Study Strategies

## Overview

The CISSP exam is notoriously challenging — not because of technical depth, but because of its breadth and the managerial mindset required. This topic covers proven study strategies, resource recommendations, and preparation timelines.

---

## 1. Study Resources

```
TOP STUDY RESOURCES:

PRIMARY STUDY GUIDES (choose 1-2):

  1. "CISSP All-in-One Exam Guide" (Shon Harris/Fernando Maymí)
     → Gold standard reference (1,300+ pages)
     → Comprehensive coverage
     → Practice questions per chapter
     → Great as reference book

  2. "CISSP Study Guide" (Eric Conrad)
     → Concise and focused (600 pages)
     → Exam-oriented content
     → Maps directly to objectives
     → Best for efficient studying

  3. "(ISC)² CISSP CBK Reference" (Official)
     → Official (ISC)² textbook
     → Most authoritative source
     → Dense, reference-style
     → Supplement with other materials

  4. "Destination CISSP" (Rob Witcher)
     → Newer, well-organized
     → Clear explanations
     → Good for beginners to CISSP content
     → Available in print and Kindle

VIDEO COURSES:
  1. Thor Pedersen (Udemy + YouTube)
     → Complete CISSP course
     → Practice exams
     → Study tips
     → Wait for Udemy sales
  
  2. Kelly Handerhan (Cybrary)
     → "Why you will pass CISSP"
     → Focus on mindset
     → Free on Cybrary
     → Watch her motivational videos
  
  3. Mike Chapple (LinkedIn Learning)
     → Domain-by-domain coverage
     → Professional quality
     → Part of LinkedIn subscription

  4. Destination CISSP (YouTube/Podcast)
     → Free podcast format
     → Bite-sized lessons
     → Good for commute study
```

  → 45 min: Read study guide
  → 30 min: Watch video course
  → 30 min: Practice questions (25-30)
  → 15 min: Review incorrect answers
  → 15 min: Flashcards/terminology
  → 15 min: Summarize key concepts

PRACTICE EXAM RESOURCES:
  1. (ISC)² Official Practice Tests (Wiley)
  2. Boson ExSim CISSP
  3. Thor Pedersen practice exams
  4. CCCure (free tier available)

TARGET PRACTICE SCORES:
  First attempt: 60-70% (expected)
  After study: 75-80%
  Exam ready: 80-85%+

    are right/wrong, not memorization

## 3. Domain-Specific Tips

```
DOMAIN STUDY TIPS:

D1 - SECURITY & RISK MANAGEMENT (15%):
  → Most important domain — study first
  → Master quantitative risk (ALE, SLE, ARO)
  → Know (ISC)² Code of Ethics
  → Understand BCP/DRP phases
  → Know legal concepts (due diligence/care)
  → Security governance roles

D2 - ASSET SECURITY (10%):
  → Data classification levels
  → Data roles (owner, custodian, steward)
  → Privacy regulations (GDPR, HIPAA)
  → Data remanence concepts
  → Shortest domain — don't skip it

D3 - SECURITY ARCHITECTURE (13%):
  → Security models are heavily tested:
    Bell-LaPadula, Biba, Clark-Wilson,
    Brewer-Nash (Chinese Wall)
  → Common Criteria evaluation levels
  → Cryptography fundamentals
  → System architecture concepts

D4 - NETWORK SECURITY (13%):
  → OSI model layers and protocols
  → Know network attacks well
  → VPN protocols (IPSec phases)
  → Firewall types
  → Wireless security protocols

D5 - IAM (13%):
  → Access control models (know cold)
  → Authentication factors
  → Kerberos process
  → Federation concepts
  → Provisioning lifecycle

D6 - ASSESSMENT & TESTING (12%):
  → Assessment vs audit vs pen test
  → SOC report types (1, 2, 3)
  → Vulnerability management
  → Software testing methods

D7 - SECURITY OPERATIONS (13%):
  → Incident response phases
  → Digital forensics process
  → Evidence handling
  → DR/BC planning phases
  → Change management

D8 - SOFTWARE DEVELOPMENT (11%):
  → SDLC phases
  → OWASP Top 10
  → Database security concepts
  → DevSecOps principles
  → Software testing types
```

## 5. Final Preparation Checklist

```
2 WEEKS BEFORE EXAM:

  [ ] Completed all domain study
  [ ] Scoring 80%+ on practice exams
  [ ] Reviewed all weak domains
  [ ] Watched Kelly Handerhan's mindset videos
  [ ] Reviewed (ISC)² Code of Ethics
  [ ] Memorized key formulas (ALE, SLE, ARO)
  [ ] Know all access control models
  [ ] Know all security models
  [ ] Understand BCP/DRP phases
  [ ] Can explain risk management concepts

1 WEEK BEFORE:
  [ ] Take final practice exam (full length)
  [ ] Review any remaining weak areas
  [ ] Light review only (no new material)
  [ ] Verify exam appointment
  [ ] Prepare ID documents
  [ ] Get good sleep all week

DAY BEFORE:
  [ ] No studying (or very light only)
  [ ] Prepare clothes and food
  [ ] Set alarms
  [ ] Review high-level concepts briefly
  [ ] Relax and rest

EXAM DAY:
  [ ] Good breakfast
  [ ] Arrive early
  [ ] Two forms of ID
  [ ] Manage time (1.6 min/question)
  [ ] Read every question twice
  [ ] THINK LIKE A MANAGER
  [ ] Safety of life always first
  [ ] Don't change initial answers
  [ ] Take deep breaths if stressed
  [ ] Trust your preparation
```
