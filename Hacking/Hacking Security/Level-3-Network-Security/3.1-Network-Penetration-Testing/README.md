# 3.1 Network Penetration Testing

## Level 3: Network Security

---

## 📋 Course Overview

Network Penetration Testing focuses on practical techniques for identifying and exploiting vulnerabilities in network infrastructure. This subject covers everything from initial network scanning to advanced evasion techniques, with hands-on tool commands, lab scenarios, and defensive countermeasures.

## 🎯 Learning Objectives

By the end of this subject, you will be able to:

- Perform comprehensive network scanning and enumeration
- Identify and exploit common network vulnerabilities
- Execute password attacks using industry-standard tools
- Understand and perform Man-in-the-Middle attacks
- Attack common network protocols (SMB, LDAP, Kerberos, SNMP)
- Test wireless network security (WPA/WPA2, Evil Twin)
- Use Metasploit for network exploitation
- Pivot through compromised networks
- Evade network security controls (IDS/IPS, firewalls)
