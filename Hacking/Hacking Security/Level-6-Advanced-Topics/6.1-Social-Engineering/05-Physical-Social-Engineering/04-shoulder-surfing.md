# Unit 5: Physical Social Engineering — Topic 4: Shoulder Surfing

## Overview

**Shoulder surfing** is the practice of observing someone's screen, keyboard, or documents to capture sensitive information such as passwords, PINs, credit card numbers, or confidential data. This low-tech attack can be performed in-person or using technology like cameras and binoculars. In the modern open-office environment, shoulder surfing is easier than ever.

---

## 1. How Shoulder Surfing Works

```
SHOULDER SURFING SCENARIOS:

IN THE OFFICE:
  → Open floor plans with no privacy screens
  → Looking over cubicle walls
  → Walking past desks and glancing at screens
  → Standing behind someone at a workstation
  → Observing during presentations/meetings

AT ATMs AND POS TERMINALS:
  → Standing behind someone entering PIN
  → Using miniature cameras
  → Watching from a distance with binoculars
  → Positioned mirrors
  → Overhead cameras in ceilings

IN PUBLIC SPACES:
  → Coffee shops (working on laptops)
  → Airports (typing passwords on WiFi login)
  → Public transit (scrolling through emails)
  → Hotel lobbies (checking sensitive documents)
  → Co-working spaces

DURING PHONE CALLS:
  → Overhearing PIN recitation
  → Listening to account verification
  → Credit card number read aloud
  → Sensitive business discussions
  → Speakerphone conversations
```

---

## 2. What Can Be Captured

```
INFORMATION TARGETS:

LOGIN CREDENTIALS:
  → Watch typing pattern for passwords
  → Observe username on login screens
  → Note password length and character positions
  → Multiple observations to build full password
  → Most effective when typing is slow

FINANCIAL DATA:
  → PIN numbers at ATMs
  → Credit card numbers during transactions
  → Account numbers on screens
  → Wire transfer details
  → Banking login credentials

CONFIDENTIAL INFORMATION:
  → Emails on screen
  → Documents being reviewed
  → Spreadsheets with sensitive data
  → Chat messages with colleagues
  → Calendar with meeting details

AUTHENTICATION CODES:
  → 2FA codes displayed on screen or phone
  → Temporary passwords
  → Security questions being answered
  → Badge/access codes entered on keypads
  → WiFi passwords entered
```

## 3. Advanced Shoulder Surfing Techniques

```
TECHNOLOGY-ENHANCED:

MINIATURE CAMERAS:
  → Pinhole cameras near ATMs
  → Camera in hat/glasses
  → Phone camera (pretend to text)
  → Wi-Fi connected cameras for remote viewing
  → Endoscope cameras for tight spaces

LONG-RANGE OBSERVATION:
  → Binoculars from adjacent building
  → Telescope for window observation
  → High-zoom camera from parking lot
  → Drone with camera (outdoor scenarios)

RECORDING TECHNIQUES:
  → Video record keyboard while "checking phone"
  → Screen recording malware (digital equivalent)
  → Photos of whiteboards after meetings
  → Video of presentations with sensitive data

REFLECTION TECHNIQUES:
  → Read screens reflected in glasses/windows
  → Use mirrors positioned strategically
  → Reflective surfaces near targets
  → Sunglasses for concealed observation

KEYSTROKE INFERENCE:
  → Watch which keys are pressed
  → Audio analysis of keystroke sounds
  → Different keys make slightly different sounds
  → Machine learning can infer keystrokes from audio
  → Accelerometer data from nearby phones
```

## 4. Environmental Factors

```
HIGH-RISK ENVIRONMENTS:

OPEN OFFICES:
  Risk Level: HIGH
  → No physical barriers between desks
  → Shared workspaces
  → Standing desks at eye level
  → Visitors and guests walking through
  → Large monitors visible from distance

PUBLIC SPACES:
  Risk Level: VERY HIGH
  → No expectation of privacy
  → Strangers in close proximity
  → Crowded environments
  → Free WiFi login screens
  → No organizational controls

CONFERENCES/EVENTS:
  Risk Level: HIGH
  → Shared tables and spaces
  → Competitors present
  → WiFi login visible
  → Presentations with sensitive info
  → Networking involves close proximity

TRAVEL:
  Risk Level: HIGH
  → Airplane neighbors can see screens
  → Hotel lobbies and business centers
  → Taxi/rideshare drivers can observe
  → Train commuters in close quarters
  → Airport lounges
```

---

## 5. Defending Against Shoulder Surfing

```
PERSONAL DEFENSES:
  → Be aware of surroundings
  → Shield PIN entry with hand
  → Position screen away from passersby
  → Sit with back to wall
  → Use password managers (no typing)
  → Biometric authentication where possible
  → Reduce screen brightness in public

PHYSICAL CONTROLS:
  → Privacy screens/filters on monitors
  → Privacy film on laptop screens
  → High cubicle walls or offices
  → Screen positioning away from walkways
  → Frosted glass for meeting rooms
  → CCTV to deter and detect

DIGITAL CONTROLS:
  → Auto-lock screens (short timeout)
  → Password masking (show dots, not characters)
  → Biometric login (fingerprint, face)
  → Smart card + PIN (two factors)
  → Screen dimming in sensitive apps
  → Virtual keyboard for sensitive input

ORGANIZATIONAL:
  → Clean desk policy
  → Screen lock policy (Win+L / Cmd+L)
  → Privacy screen requirement for travel
  → Sensitive work in private areas only
  → Security awareness training
  → "Lock it when you leave it" culture
```
