# Unit 5: Physical Social Engineering — Topic 2: Impersonation

## Overview

**Impersonation** is the act of assuming a false identity to gain unauthorized access, information, or trust. It is one of the most versatile physical social engineering techniques, combining pretexting with props, clothing, and behavior to convincingly portray someone with a legitimate reason to be on premises. Effective impersonation can bypass security guards, reception, and even badge-controlled access.

## 1. Common Impersonation Roles

```
HIGH-SUCCESS IMPERSONATION ROLES:

MAINTENANCE/FACILITIES:
  → HVAC technician
  → Electrician / Plumber
  → Janitorial staff
  → Elevator repair technician
  → Fire suppression system inspector
  WHY: Expected to access restricted areas

IT/TELECOM:
  → Cable installer
  → Network technician
  → Copier/printer repair
  → Security camera installer
  → Phone system technician
  WHY: Expected to access server rooms/closets

DELIVERY:
  → Package delivery (FedEx, UPS, etc.)
  → Food delivery (catering, lunch orders)
  → Office supply delivery
  → Equipment delivery
  WHY: Routinely allowed in without ID check

PROFESSIONAL:
  → Auditor / Inspector
  → Job candidate
  → Consultant
  → Photographer
  → Real estate appraiser
  WHY: Business-related reason for access

AUTHORITY:
  → Fire marshal
  → Health inspector
  → Building inspector
  → Insurance adjuster
  → Law enforcement (ILLEGAL if not actual LE)
  WHY: High compliance due to authority
```

## 2. Impersonation Preparation

```
PREPARATION CHECKLIST:

RESEARCH:
  □ Company's current vendors/contractors
  □ Recent work orders or projects
  □ Names of facilities/IT managers
  □ Building layout and access points
  □ Security measures in place
  □ Employee dress code and culture

APPEARANCE:
  □ Appropriate clothing/uniform
  □ Role-specific tools and equipment
  □ Company-branded items (if impersonating vendor)
  □ Badge/lanyard (generic or cloned)
  □ Clipboard with official-looking paperwork
  □ Vehicle with appropriate markings (advanced)

DOCUMENTATION:
  □ Fake work order with real details
  □ Business cards with callback number
  □ Official-looking forms
  □ Email confirmation printout
  □ Vendor agreement (if applicable)

COMMUNICATION:
  □ Callback phone number (your control)
  □ Email address matching claimed identity
  □ Name of person who "scheduled" the visit
  □ Reference to specific work/project
  □ Knowledge of company-specific terminology

BEHAVIOR:
  □ Practice the role's mannerisms
  □ Know common tasks for the role
  □ Prepared for verification questions
  □ Exit strategy if confronted
  □ Backup story if primary fails
```

## 3. Impersonation Execution

```
EXECUTION FRAMEWORK:

ARRIVAL:
  → Arrive at appropriate time for the role
  → Park in correct area (service entrance for vendor)
  → Walk with purpose and confidence
  → Have props visible and ready

FIRST CONTACT:
  → State identity clearly and confidently
  → "Hi, I'm [Name] from [Company], here for 
     the scheduled [maintenance/installation]"
  → Show work order or documentation
  → Reference a real person who "arranged" the visit

VERIFICATION HANDLING:
  → Offer to have them call "your office"
     (number you control)
  → Reference specific internal contacts by name
  → Show business cards, documentation
  → Appear mildly inconvenienced but understanding
  → "Sure, check with [name in facilities]"

NAVIGATION:
  → Ask for escort if expected for role
  → If confident, navigate independently
  → Use small talk with employees along the way
  → "I've been here before — the server room is 
     down this hall, right?"
  → Correct information = builds credibility

OBJECTIVE COMPLETION:
  → Accomplish actual goal while "working"
  → Install rogue device while "checking cables"
  → Take photos while "inspecting equipment"
  → Gather info while making conversation
  → Plant USB drives while "moving equipment"

EXIT:
  → Thank reception/security on the way out
  → "Everything looks good — should be all set"
  → Sign out if required (fake name)
  → Leave naturally, don't rush
  → Follow up with thank-you email (optional depth)
```

## 4. Advanced Impersonation Tactics

```
BADGE CLONING:
  → Capture badge data from proximity cards
  → Clone onto blank card
  → Requires RFID reader (Proxmark, HID reader)
  → Works on 125kHz (HID Prox, EM4100)
  → Harder with 13.56MHz (iCLASS, DESFire)

UNIFORM ACQUISITION:
  → Purchase online (security guard, maintenance)
  → Thrift stores (work shirts with logos)
  → Screen print custom logos
  → Safety vest + hard hat = universal access
  → Pizza delivery shirt + insulated bag

VEHICLE STAGING:
  → Magnetic signs for van/truck
  → "ABC Plumbing" or "XYZ Electric"
  → Adds significant credibility
  → Park in service/loading areas
  → Legitimate-looking work vehicle

MULTI-PERSON TEAMS:
  → Two people are more believable than one
  → "Senior tech + apprentice" dynamic
  → One distracts while other works
  → Mutual validation ("we're together")
  → Harder for security to challenge a team
```

## 5. Defending Against Impersonation

```
DEFENSE MEASURES:

VISITOR MANAGEMENT:
  → All visitors pre-registered
  → Photo ID required and retained
  → Visitor badges clearly marked
  → Escort required at all times
  → Sign-in/sign-out log

VENDOR VERIFICATION:
  → Maintain approved vendor list
  → Verify work orders before granting access
  → Call vendor's main number (not number given)
  → Require advance scheduling
  → Escort vendors in sensitive areas

EMPLOYEE TRAINING:
  → Challenge unknown persons politely
  → Verify identity through official channels
  → Report unescorted visitors
  → "If you see something, say something"
  → Regular impersonation drills

ACCESS CONTROLS:
  → Badge required everywhere
  → Different badge levels (staff vs visitor)
  → Anti-tailgating at key checkpoints
  → CCTV at all entrances
  → Visitor badge expiration
```
