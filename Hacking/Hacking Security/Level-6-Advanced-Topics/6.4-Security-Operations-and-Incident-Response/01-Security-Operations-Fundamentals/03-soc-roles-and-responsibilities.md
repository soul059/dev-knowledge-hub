# Unit 1: Security Operations Fundamentals вҖ” Topic 3: SOC Roles and Responsibilities

## Overview

A well-functioning SOC requires clearly defined **roles and responsibilities** organized in a tiered structure. Each role has specific skills, duties, and career progression paths. Understanding the SOC organizational structure is essential for building effective teams and developing security career paths.

## 2. Tier 1: SOC Analyst (Junior)

```
TIER 1 SOC ANALYST:

  TITLE: SOC Analyst I / Security Monitor / Alert Analyst

  PRIMARY DUTIES:
  вҶ’ Monitor security dashboards and alerts
  вҶ’ Initial alert triage (true positive vs false positive)
  вҶ’ Follow standard operating procedures (SOPs)
  вҶ’ Execute basic playbook steps
  вҶ’ Escalate confirmed incidents to Tier 2
  вҶ’ Create and update tickets
  вҶ’ Basic log review and analysis
  вҶ’ Documentation of findings

  DAILY WORKFLOW:
  1. Check SIEM dashboard at shift start
  2. Review and triage new alerts
  3. Follow playbook for each alert type
  4. Document analysis in ticket
  5. Escalate if confirmed or uncertain
  6. Close false positives with notes
  7. End-of-shift handoff report

  REQUIRED SKILLS:
  вҶ’ Understanding of networking (TCP/IP, DNS, HTTP)
  вҶ’ Basic knowledge of operating systems
  вҶ’ Familiarity with SIEM tools
  вҶ’ Understanding of common attack types
  вҶ’ Log analysis basics
  вҶ’ Strong documentation skills
  вҶ’ Attention to detail

  CERTIFICATIONS:
  вҶ’ CompTIA Security+
  вҶ’ CompTIA CySA+
  вҶ’ Splunk Core Certified User
  вҶ’ GIAC Security Essentials (GSEC)

  EXPERIENCE: 0-2 years
  SALARY RANGE: $50,000-$75,000
  
  ESCALATION CRITERIA:
  вҶ’ Confirmed malware execution
  вҶ’ Successful exploitation
  вҶ’ Data exfiltration indicators
  вҶ’ Unauthorized access
  вҶ’ Alert outside playbook scope
  вҶ’ Uncertain analysis
```

## 3. Tier 2: SOC Analyst (Senior)

```
TIER 2 SOC ANALYST:

  TITLE: SOC Analyst II / Incident Responder / Security Analyst

  PRIMARY DUTIES:
  вҶ’ Deep-dive investigation of escalated alerts
  вҶ’ Determine scope and impact of incidents
  вҶ’ Perform root cause analysis
  вҶ’ Execute containment actions
  вҶ’ Coordinate with IT for remediation
  вҶ’ Develop and refine detection rules
  вҶ’ Mentor Tier 1 analysts
  вҶ’ Write incident reports
  вҶ’ Conduct basic threat hunting

  INVESTIGATION PROCESS:
  1. Receive escalation from Tier 1
  2. Validate and expand investigation
  3. Correlate across multiple data sources
  4. Determine attack scope
  5. Execute containment if needed
  6. Document findings and timeline
  7. Coordinate remediation
  8. Escalate to Tier 3 if needed

  REQUIRED SKILLS:
  вҶ’ Advanced log analysis
  вҶ’ Malware analysis basics
  вҶ’ Network forensics
  вҶ’ Endpoint forensics
  вҶ’ Scripting (Python, PowerShell)
  вҶ’ Advanced SIEM queries (SPL, KQL)
  вҶ’ Incident response methodology
  вҶ’ Threat intelligence consumption
  вҶ’ Strong communication skills

  CERTIFICATIONS:
  вҶ’ GIAC Certified Incident Handler (GCIH)
  вҶ’ CompTIA CySA+
  вҶ’ EC-Council ECIH
  вҶ’ Splunk Certified Power User
  вҶ’ SANS FOR508

  EXPERIENCE: 2-5 years
  SALARY RANGE: $75,000-$110,000
```
