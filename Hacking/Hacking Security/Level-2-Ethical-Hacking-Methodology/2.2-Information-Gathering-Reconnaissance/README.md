# 2.2 Information Gathering and Reconnaissance

## 📋 Course Overview

This subject covers the **reconnaissance phase** of penetration testing — the critical first step where information about the target is gathered before any exploitation. From passive OSINT techniques to active network scanning, these notes provide a complete guide to information gathering methodology, tools, and automation.
