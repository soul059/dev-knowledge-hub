# 📱 Subject 5.2: Mobile Application Security

## Course Overview

This subject provides comprehensive coverage of **mobile application security** for both Android and iOS platforms. You'll learn mobile security architectures, testing methodologies, common vulnerabilities, and secure development practices. The content aligns with the **OWASP Mobile Security Testing Guide (MSTG)** and **OWASP Mobile Top 10**.

## 🎯 Learning Objectives

By completing this subject, you will be able to:

- Understand mobile security architectures for Android and iOS
- Set up mobile security testing environments
- Perform static and dynamic analysis of mobile applications
- Identify and exploit common mobile vulnerabilities
- Test mobile API security
- Bypass security controls (root/jailbreak detection, certificate pinning)
- Analyze mobile malware
- Apply OWASP MSTG testing methodology
