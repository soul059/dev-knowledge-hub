# 📡 Networking for Security Professionals

## Subject 1.2 | Level 1 - Foundations

---

## Course Overview

This subject covers networking concepts from a **security perspective**. Understanding how networks work — and how they can be attacked — is fundamental to every cybersecurity role. Each topic includes network diagrams, packet analysis concepts, command examples, attack surfaces, and security controls.

## How to Use These Notes

1. **Sequential Learning**: Follow units in order (Layer 2 → Layer 3 → Layer 4 → Application)
2. **Lab Practice**: Try commands in a safe lab environment
3. **Capture Traffic**: Use Wireshark to reinforce protocol concepts
4. **Think Like an Attacker**: Understand attacks to build better defenses
