## 3. Key Roles Explained

### Offensive Security Roles (Red Team)

| Role | Description | Key Skills | Certifications |
|------|-------------|------------|----------------|
| **Penetration Tester** | Tests systems for vulnerabilities | Networking, scripting, exploit dev | OSCP, CEH, PNPT |
| **Red Team Operator** | Simulates real-world attacks end-to-end | Advanced exploitation, social eng | CRTO, OSEP, OSCE |
| **Bug Bounty Hunter** | Finds vulns in exchange for bounties | Web app security, creativity | — |
| **Exploit Developer** | Creates proof-of-concept exploits | Assembly, reverse engineering | OSED, GXPN |
| **Vulnerability Researcher** | Discovers zero-day vulnerabilities | Deep technical skills, patience | GREM, specialized |

### Defensive Security Roles (Blue Team)

| Role | Description | Key Skills | Certifications |
|------|-------------|------------|----------------|
| **SOC Analyst** | Monitors and triages security alerts | SIEM, log analysis, networking | Security+, CySA+, BTL1 |
| **Incident Responder** | Handles security incidents | Forensics, IR methodology | GCIH, ECIH |
| **Threat Hunter** | Proactively searches for threats | Threat intelligence, analytics | GCTI, CTIA |
| **Malware Analyst** | Reverse engineers malicious software | Assembly, reverse engineering | GREM |
| **Digital Forensics** | Investigates cyber crimes | Evidence handling, legal knowledge | GCFE, EnCE, CHFI |

### Governance, Risk & Compliance Roles

| Role | Description | Key Skills | Certifications |
|------|-------------|------------|----------------|
| **Security Auditor** | Assesses compliance with standards | Audit methodology, frameworks | CISA, ISO 27001 |
| **Risk Analyst** | Evaluates and manages security risks | Risk assessment, business analysis | CRISC, CISM |
| **Compliance Officer** | Ensures regulatory compliance | Legal knowledge, policy writing | CIPP, CISSP |
| **CISO** | Leads entire security program | Leadership, strategy, business | CISSP, CISM |

## 5. Getting Started — Entry Roadmap

```
STEP 1: Build Foundations
   │
   ├── Learn networking (CompTIA Network+)
   ├── Learn Linux basics
   ├── Learn basic programming (Python)
   └── Understand security concepts
   │
   ▼
STEP 2: Get Certified
   │
   ├── CompTIA Security+  (industry standard entry cert)
   ├── OR  BTL1 / eJPT   (hands-on alternatives)
   └── Start building a home lab
   │
   ▼
STEP 3: Gain Experience
   │
   ├── CTF competitions (TryHackMe, HackTheBox)
   ├── Personal projects and labs
   ├── IT support / help desk roles
   ├── Contribute to open-source security tools
   └── Internships
   │
   ▼
STEP 4: Specialize
   │
   ├── Choose: Offensive / Defensive / GRC / Engineering
   ├── Get specialized certifications
   └── Build expertise through practice
   │
   ▼
STEP 5: Advance
   │
   ├── Senior roles and leadership
   ├── Mentor others
   └── Continuous professional development
```
