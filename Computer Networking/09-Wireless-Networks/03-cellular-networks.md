## â“ Quick Revision Questions

1. **Q:** What is frequency reuse in cellular networks?
   <details>
   <summary>Answer</summary>
   **Frequency Reuse**:
   
   Same frequencies used in cells far enough apart to avoid interference.
   
   **Why it matters**:
   - Limited spectrum available
   - Many users need service
   - Reuse multiplies capacity
   
   **Reuse Factor**: Number of cells before frequency repeats (commonly 7).
   
   Cells using same frequency are separated by several cells in between.
   </details>

2. **Q:** Compare TDMA, CDMA, and OFDMA.
   <details>
   <summary>Answer</summary>
   **TDMA** (Time Division):
   - Divide time into slots
   - Each user gets a time slot
   - Used in: 2G GSM
   
   **CDMA** (Code Division):
   - All users transmit simultaneously
   - Unique code per user
   - Receiver extracts using code
   - Used in: 3G UMTS
   
   **OFDMA** (Orthogonal Frequency):
   - Many subcarriers
   - Users allocated subcarriers
   - Flexible allocation
   - Used in: 4G LTE, 5G
   </details>

3. **Q:** What are the three 5G service categories?
   <details>
   <summary>Answer</summary>
   **eMBB** (Enhanced Mobile Broadband):
   - High speed (10 Gbps+)
   - 4K/8K video, VR, AR
   
   **URLLC** (Ultra Reliable Low Latency):
   - Sub-millisecond latency
   - 99.999% reliability
   - Autonomous vehicles, remote surgery
   
   **mMTC** (Massive Machine Type Comm):
   - 1 million devices per kmÂ²
   - Low power, small packets
   - IoT, smart cities, sensors
   </details>

   <details>
   <summary>Answer</summary>
   **Handoff**: Transferring connection from one cell to another while moving.
   
   **Hard Handoff**:
   - Break-before-make
   - Disconnect from old, connect to new
   - Brief interruption possible
   - GSM, LTE
   
   **Soft Handoff**:
   - Make-before-break
   - Connected to both cells temporarily
   - Smoother transition
   - CDMA only
   </details>

5. **Q:** Why does 5G use mmWave frequencies?
   <details>
   <summary>Answer</summary>
   **mmWave** (24-100 GHz):
   
   **Advantages**:
   - Massive bandwidth available
   - Very high speeds (10+ Gbps)
   - Unused spectrum
   
   **Challenges**:
   - Very short range
   - Blocked by walls, trees, rain
   - Requires many small cells
   
   **Solution**: Dense deployment + beamforming + lower frequencies for coverage.
   </details>

6. **Q:** What is network slicing in 5G?
   <details>
   <summary>Answer</summary>
   **Network Slicing**: Creating multiple virtual networks on same physical infrastructure.
   
   **Each slice optimized for specific use**:
   - eMBB slice: Maximum bandwidth
   - URLLC slice: Minimum latency
   - IoT slice: Maximum connections
   
   **Benefits**:
   - Different QoS guarantees
   - Resource optimization
   - Isolated failures
   - Custom policies per slice
   
   Like having multiple specialized networks without building separate infrastructure.
   </details>
