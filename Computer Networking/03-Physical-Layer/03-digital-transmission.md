## â“ Quick Revision Questions

1. **Q:** What is the main advantage of Manchester encoding over NRZ-L?
   <details>
   <summary>Answer</summary>
   The main advantage of Manchester encoding is **self-clocking**. It guarantees a transition in the middle of every bit interval, which allows the receiver to synchronize its clock with the incoming signal. NRZ-L lacks this feature and can lose synchronization during long runs of the same bit.
   </details>

2. **Q:** Why does AMI encoding have problems with long runs of 0s?
   <details>
   <summary>Answer</summary>
   In AMI encoding, 0s are represented by zero voltage (no pulse). When there are many consecutive 0s, there are no signal transitions, which means the receiver has no way to synchronize its clock with the transmitter. This can lead to timing drift and bit errors. Solutions like B8ZS and HDB3 address this by substituting patterns for long runs of 0s.
   </details>

3. **Q:** What is the purpose of block coding like 4B/5B?
   <details>
   <summary>Answer</summary>
   Block coding adds redundancy to the data stream to:
   - **Ensure adequate transitions** for clock synchronization
   - **Detect errors** (invalid codes indicate transmission errors)
   - **Avoid long runs of 0s or 1s** that cause sync problems
   
   In 4B/5B, only 16 of 32 possible 5-bit codes are used, and they're chosen to avoid problematic patterns.
   </details>

4. **Q:** What is the Nyquist theorem and why is it important?
   <details>
   <summary>Answer</summary>
   The Nyquist theorem states that **the sampling frequency must be at least twice the maximum frequency of the signal** (fs â‰¥ 2 Ã— fmax) to accurately reconstruct the original signal. This is important because:
   - Sampling below this rate causes **aliasing** (distortion)
   - It determines the minimum digitization requirements
   - For voice (4 kHz max), we need at least 8,000 samples/second
   </details>

5. **Q:** Calculate the bit rate for digitizing a 4 kHz voice signal with 8-bit samples.
   <details>
   <summary>Answer</summary>
   **Step 1: Find sampling rate**
   - Using Nyquist: fs â‰¥ 2 Ã— 4,000 Hz = 8,000 samples/second
   
   **Step 2: Calculate bit rate**
   - Bit Rate = Sampling Rate Ã— Bits per Sample
   - Bit Rate = 8,000 Ã— 8 = **64,000 bps = 64 kbps**
   
   This is the standard bit rate for digital telephone (DS0 channel).
   </details>

6. **Q:** What are the three steps in Pulse Code Modulation (PCM)?
   <details>
   <summary>Answer</summary>
   The three steps in PCM are:
   1. **Sampling** - Taking instantaneous values of the analog signal at regular intervals (following Nyquist theorem)
   2. **Quantization** - Rounding each sample value to the nearest discrete level (introduces quantization error)
   3. **Encoding** - Converting each quantized level to a binary code (the number of bits determines the number of levels)
   </details>
