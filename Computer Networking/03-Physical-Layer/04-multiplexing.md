## â“ Quick Revision Questions

1. **Q:** What is the main difference between FDM and TDM?
   <details>
   <summary>Answer</summary>
   **FDM (Frequency Division Multiplexing)** divides the available bandwidth into multiple frequency bands, with each user getting a dedicated frequency range simultaneously. Used mainly for analog signals.
   
   **TDM (Time Division Multiplexing)** divides time into slots, with each user getting the full bandwidth but only during their assigned time slot. Used for digital signals.
   </details>

2. **Q:** Why is statistical TDM more efficient than synchronous TDM?
   <details>
   <summary>Answer</summary>
   Statistical TDM is more efficient because it **dynamically allocates time slots** only to users who have data to send. In synchronous TDM, slots are reserved for each user even if they have no data, wasting bandwidth. Statistical TDM can achieve higher efficiency by allowing the link speed to be less than the sum of all input speeds.
   </details>

3. **Q:** What is the bit rate of a T1 line and how is it calculated?
   <details>
   <summary>Answer</summary>
   A T1 line has a bit rate of **1.544 Mbps**.
   
   Calculation:
   - 24 voice channels Ã— 8 bits per sample = 192 data bits
   - 1 framing bit per frame = 1 bit
   - Total: 193 bits per frame
   - 8,000 frames per second (Nyquist rate for voice)
   - 193 Ã— 8,000 = 1,544,000 bps = 1.544 Mbps
   </details>

4. **Q:** What is WDM and why is it called "FDM for optical fiber"?
   <details>
   <summary>Answer</summary>
   **WDM (Wavelength Division Multiplexing)** combines multiple light signals at different wavelengths (colors) onto a single optical fiber. It's called "FDM for optical fiber" because just as FDM divides the electromagnetic spectrum into frequency bands, WDM divides the light spectrum into different wavelengths. Each wavelength is essentially a different frequency of light, carrying independent data streams.
   </details>

5. **Q:** How does CDMA allow multiple users to transmit simultaneously on the same frequency?
   <details>
   <summary>Answer</summary>
   CDMA assigns each user a unique **chip code** (orthogonal sequence). Each user multiplies their data by their code before transmitting. At the receiver, multiplying by the same code extracts the original data while other users' signals (multiplied by different codes) cancel out due to the orthogonal property of the codes. It's like multiple conversations in different languages at a partyâ€”each listener understands only their language.
   </details>

6. **Q:** What is the difference between CWDM and DWDM?
   <details>
   <summary>Answer</summary>
   **CWDM (Coarse WDM)**: Wide channel spacing (~20 nm), supports up to 18 channels, lower cost, shorter distances, used in metro/access networks.
   
   **DWDM (Dense WDM)**: Very narrow spacing (~0.8 nm or less), supports 40-160+ channels, higher cost, longer distances, used for long-haul backbone networks. DWDM can achieve terabits per second on a single fiber.
   </details>
