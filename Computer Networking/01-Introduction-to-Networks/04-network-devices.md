### Types of Gateways

| Type | Function | Example |
|------|----------|---------|
| **Protocol Gateway** | Converts between protocols | TCP/IP to SNA |
| **Application Gateway** | Converts application data | Email gateway |
| **VoIP Gateway** | Analog to digital voice | Phone to IP |
| **Cloud Gateway** | On-premise to cloud | Storage gateway |
| **IoT Gateway** | Sensor protocols to IP | ZigBee to WiFi |

### Gateway vs Router

| Aspect | Router | Gateway |
|--------|--------|---------|
| Function | Routes between same protocol networks | Translates between different protocols |
| OSI Layer | Layer 3 | Layers 3-7 |
| Processing | Address-based routing | Full protocol conversion |
| Complexity | Moderate | High |
| Example | Internet router | VoIP gateway |

## â“ Quick Revision Questions

1. **Q:** What is the main difference between a hub and a switch?
   <details>
   <summary>Answer</summary>
   A hub broadcasts all incoming data to all ports (Layer 1, unintelligent), while a switch forwards data only to the destination port based on MAC addresses (Layer 2, intelligent). Switches provide better performance and separate collision domains.
   </details>

2. **Q:** At which OSI layer does a router operate and what address does it use?
   <details>
   <summary>Answer</summary>
   A router operates at Layer 3 (Network Layer) and uses IP addresses to make routing decisions. It connects different networks and separates broadcast domains.
   </details>

3. **Q:** What is the function of a NIC's MAC address?
   <details>
   <summary>Answer</summary>
   The MAC address is a unique 48-bit hardware identifier that uniquely identifies the NIC on a local network. It's used for Layer 2 (Data Link) addressing and is essential for switches and bridges to forward frames correctly.
   </details>

4. **Q:** How does a gateway differ from a router?
   <details>
   <summary>Answer</summary>
   A router connects networks using the same protocol (e.g., IP), while a gateway translates between different network architectures or protocols (e.g., TCP/IP to SNA). Gateways operate at higher layers (up to Layer 7) and perform full protocol conversion.
   </details>

5. **Q:** What happens when a switch receives a frame with an unknown destination MAC address?
   <details>
   <summary>Answer</summary>
   The switch floods (broadcasts) the frame to all ports except the source port. When the destination device responds, the switch learns its MAC address and port association, adding it to the MAC address table for future reference.
   </details>

6. **Q:** Why are repeaters needed in networks?
   <details>
   <summary>Answer</summary>
   Signals weaken (attenuate) as they travel over distance. Repeaters regenerate and amplify the signal to extend the transmission distance beyond the cable's normal limit. They operate at Layer 1 and don't interpret the data.
   </details>
