### What is the OSI Model?

The **OSI Reference Model** is a conceptual framework developed by the International Organization for Standardization (ISO) in 1984. It divides network communication into seven abstraction layers.

## â“ Quick Revision Questions

1. **Q:** Name all seven layers of the OSI model from bottom to top.
   <details>
   <summary>Answer</summary>
   Physical, Data Link, Network, Transport, Session, Presentation, Application (Mnemonic: "Please Do Not Throw Sausage Pizza Away")
   </details>

2. **Q:** At which layer does a router operate and why?
   <details>
   <summary>Answer</summary>
   A router operates at Layer 3 (Network Layer) because it uses IP addresses (logical addresses) to make routing decisions and forward packets between different networks.
   </details>

3. **Q:** What is the data unit at Layer 2, and what information does it contain?
   <details>
   <summary>Answer</summary>
   The data unit at Layer 2 is a Frame. It contains a header with MAC addresses (source and destination), the data (encapsulated packet), and a trailer with error checking information (like CRC).
   </details>

4. **Q:** What is the main function of the Transport Layer?
   <details>
   <summary>Answer</summary>
   The Transport Layer provides end-to-end data delivery between applications, including segmentation, reliability (with TCP), flow control, and multiplexing using port numbers.
   </details>

5. **Q:** Explain the process of encapsulation as data moves from Layer 7 to Layer 1.
   <details>
   <summary>Answer</summary>
   Each layer adds its own header (and Layer 2 adds a trailer) to the data from the layer above:
   - Layer 4: Adds TCP/UDP header â†’ Segment
   - Layer 3: Adds IP header â†’ Packet
   - Layer 2: Adds frame header and trailer â†’ Frame
   - Layer 1: Converts to bits for transmission
   </details>

6. **Q:** What is the difference between the Session and Presentation layers?
   <details>
   <summary>Answer</summary>
   - Session Layer (5): Manages sessions between applications - establishes, maintains, and terminates connections, plus synchronization with checkpoints.
   - Presentation Layer (6): Handles data format translation between systems, encryption/decryption, and data compression.
   </details>
