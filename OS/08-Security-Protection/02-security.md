# Chapter 8.2: Security

## Overview

Security in operating systems involves protecting the computer system from external threats including unauthorized access, malicious attacks, and data breaches. While protection deals with internal access control mechanisms, security focuses on defending against threats from outside the system.

## Summary

| Concept | Description |
|---------|-------------|
| **CIA Triad** | Confidentiality, Integrity, Availability |
| **Virus** | Self-replicating, attaches to programs, needs user action |
| **Worm** | Self-replicating, standalone, spreads automatically |
| **Trojan** | Appears legitimate, contains hidden malware |
| **Authentication Factors** | Know (password), Have (token), Are (biometric) |
| **Symmetric Encryption** | Same key for encrypt/decrypt (AES) |
| **Asymmetric Encryption** | Public key encrypts, private key decrypts (RSA) |
| **Hashing** | One-way function, used for passwords and integrity |
| **Firewall** | Controls network traffic based on rules |
| **Sandboxing** | Isolates applications in restricted environments |
