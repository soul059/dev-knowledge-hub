|-----------|----------|

| **Instruction Register (IR)** | Holds current instruction being executed |
| **Instruction Decoder** | Interprets the opcode and generates signals |
| **Timing Generator** | Produces clock signals to synchronize operations |

|---------|------|------|

| **Refresh** | Not required | Required |

| **Use** | Cache memory | Main memory |
| **Cell** | 6 transistors | 1 transistor + capacitor |

## â“ Quick Revision Questions

1. **Q**: What is the difference between MAR and MDR?
   
   **A**: MAR (Memory Address Register) holds the memory address to be accessed, while MDR (Memory Data Register) holds the data being read from or written to memory. MAR connects to the address bus; MDR connects to the data bus.

2. **Q**: Why is SRAM faster than DRAM? Where is each used?
   
   **A**: SRAM uses flip-flops (6 transistors per bit) that don't need refreshing, while DRAM uses capacitors that need periodic refresh. SRAM is used for cache memory; DRAM is used for main memory.

3. **Q**: What are the main functions of the Control Unit?
   
   **A**: (1) Fetch instructions from memory, (2) Decode instructions, (3) Generate control signals, (4) Coordinate data flow between CPU components, (5) Manage timing and sequencing.

4. **Q**: Explain the difference between memory-mapped I/O and isolated I/O.
   
   **A**: Memory-mapped I/O shares address space with memory and uses same instructions (LOAD/STORE). Isolated I/O has separate address space and uses special instructions (IN/OUT).

5. **Q**: What is the role of the I/O interface in a computer system?
   
   **A**: The I/O interface: (1) Buffers data for speed matching, (2) Converts data formats, (3) Decodes addresses, (4) Synchronizes timing, (5) Handles interrupts, (6) Detects errors.

6. **Q**: Draw and label the ALU showing its inputs and outputs.
   
   **A**: See Section 1 - ALU receives two operands and operation select signal; outputs result and status flags (Zero, Carry, Sign, Overflow).
