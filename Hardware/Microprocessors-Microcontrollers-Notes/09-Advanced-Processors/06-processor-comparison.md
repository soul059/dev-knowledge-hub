## â“ Quick Revision Questions

1. **What is the main advantage of 80386 over 80286?**
   <details>
   <summary>Show Answer</summary>
   True 32-bit architecture (registers, ALU, bus), 4 GB physical memory, paging for virtual memory, Virtual 8086 mode for DOS compatibility, ability to exit protected mode by software, and 4 GB segment limits vs 64 KB.
   </details>

2. **Why is ARM dominant in mobile devices?**
   <details>
   <summary>Show Answer</summary>
   RISC architecture provides better performance per watt. Simpler design = fewer transistors = less power. Licensing model allows customization. Thumb mode provides good code density. Years of mobile-focused optimization.
   </details>

3. **What made Pentium significantly faster than 486?**
   <details>
   <summary>Show Answer</summary>
   Superscalar execution (2 instructions/clock vs 1), 64-bit data bus (doubles memory bandwidth), split L1 cache (simultaneous code+data access), branch prediction (reduces stalls), enhanced FPU (8Ã— faster).
   </details>

4. **When would you choose 8051 over ARM Cortex-M?**
   <details>
   <summary>Show Answer</summary>
   8051 when: extremely cost-sensitive, legacy code exists, simpler peripherals needed, bit-addressable I/O useful, lower learning curve, 8-bit sufficient. ARM when: more processing power needed, advanced peripherals, lower power, 32-bit math, modern tools preferred.
   </details>

5. **How did x86 address RISC efficiency?**
   <details>
   <summary>Show Answer</summary>
   Starting with Pentium Pro (P6), x86 decodes complex CISC instructions into simpler internal micro-operations (Âµops). These Âµops are executed like RISC instructions: out-of-order, superscalar, pipelined. Maintains backwards compatibility while gaining RISC efficiency.
   </details>

6. **What determines whether to use microprogrammed vs hardwired control?**
   <details>
   <summary>Show Answer</summary>
   Microprogrammed: complex instruction sets, flexibility to update/fix, easier design. CISC processors typically use this. Hardwired: simple instruction sets, maximum speed, fixed function, lower power. RISC processors typically use this. Modern processors combine both.
   </details>
