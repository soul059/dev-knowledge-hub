COMMON PROGRAMMING PATTERNS
â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”

1. Delay using Timer 0 Mode 1:
   MOV TMOD, #01H      ; Timer 0 Mode 1
   MOV TH0, #HIGH_VAL
   MOV TL0, #LOW_VAL
   SETB TR0            ; Start timer
   JNB TF0, $          ; Wait for overflow
   CLR TR0             ; Stop timer
   CLR TF0             ; Clear flag

2. Enable External Interrupt 0:
   SETB IT0            ; Edge triggered
   SETB EX0            ; Enable INT0
   SETB EA             ; Global enable
   
3. Serial Transmit (9600 baud):
   MOV TMOD, #20H      ; Timer 1 Mode 2
   MOV TH1, #-3        ; 9600 baud
   MOV SCON, #50H      ; Mode 1, REN=1
   SETB TR1            ; Start baud rate
   MOV SBUF, A         ; Send data
   JNB TI, $           ; Wait for complete
   CLR TI              ; Clear flag

4. Read from look-up table:
   MOV DPTR, #TABLE
   MOV A, INDEX
   MOVC A, @A+DPTR     ; A = TABLE[INDEX]
