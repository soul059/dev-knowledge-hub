### 2.2 Detailed Feature Comparison

| Feature | Microprocessor | Microcontroller |
|---------|---------------|-----------------|
| **Definition** | CPU on a chip | Computer on a chip |
| **Components** | Only CPU | CPU + Memory + I/O + Peripherals |
| **External Memory** | Required | Optional (built-in) |
| **RAM** | External (large) | Internal (small: 128B-256KB) |
| **ROM** | External | Internal (4KB-2MB) |
| **I/O Ports** | External chips | Built-in (8-32+ pins) |
| **Timers** | External (8253/54) | Built-in (1-4 timers) |
| **Serial Comm** | External (8251) | Built-in UART/SPI/I2C |
| **Interrupts** | External (8259) | Built-in controller |
| **ADC/DAC** | External | Often built-in |
| **Clock Generator** | Often external | Built-in oscillator |
| **Power Consumption** | Higher | Lower |
| **Cost of System** | Higher | Lower |
| **Board Size** | Larger | Smaller |
| **Bit Manipulation** | Limited | Excellent |
| **Processing Power** | Higher | Lower to Moderate |

## 6. Selection Criteria

### 6.1 When to Choose a Microprocessor

âœ… **Choose Microprocessor when:**
- High computational power is required
- Running complex operating systems (Windows, Linux)
- Large memory requirements (GB of RAM)
- Need for expandability and upgrades
- Multiple high-speed peripherals required
- Running multiple applications simultaneously
- Graphics and video processing
- General-purpose computing

### 6.2 When to Choose a Microcontroller

âœ… **Choose Microcontroller when:**
- Dedicated, single-purpose application
- Cost is a major factor
- Low power consumption needed
- Small form factor required
- Real-time control is essential
- Simple I/O operations
- Bit-level manipulation needed
- Mass production (cost per unit matters)

## â“ Quick Revision Questions

1. **Define microprocessor and microcontroller. What is the fundamental difference?**
   <details>
   <summary>Show Answer</summary>
   A microprocessor contains only the CPU on a chip (CPU on a chip), requiring external memory and I/O. A microcontroller contains CPU, memory, I/O ports, and peripherals all on a single chip (computer on a chip).
   </details>

2. **Why is a microcontroller called "computer on a chip"?**
   <details>
   <summary>Show Answer</summary>
