## ğŸ“Š Quick Comparison: Microprocessor vs Microcontroller

| Feature | Microprocessor | Microcontroller |
|---------|---------------|-----------------|
| Definition | CPU on a chip | Computer on a chip |
| Memory | External | Internal + External |
| I/O Ports | External | Built-in |
| Cost | Higher system cost | Lower system cost |
| Applications | General purpose computing | Embedded systems |
