## Course Overview

Differential Equations form the mathematical backbone of modeling real-world phenomena â€” from population growth and radioactive decay to electrical circuits and mechanical vibrations. This course covers **Ordinary Differential Equations (ODEs)** systematically, building from first-order equations to systems, series solutions, and transform methods.

### What You Will Learn

| Area | Key Skills |
|------|-----------|
| **Foundations** | Classify ODEs by order, degree, linearity; form DEs from families of curves |
| **First Order Methods** | Solve separable, homogeneous, linear, exact, and Bernoulli equations |
| **Higher Degree** | Handle Clairaut's and Lagrange's equations; find singular solutions |
| **Applications** | Model circuits (RL, RC, LCR), oscillations, growth/decay, trajectories |
| **Higher Order** | Solve constant-coefficient ODEs; use Wronskian and reduction of order |
| **Non-Homogeneous** | Apply undetermined coefficients, variation of parameters, D-operator |
| **Systems** | Convert to first-order systems; use matrix methods and phase planes |
| **Series Solutions** | Power series, Frobenius method, Bessel and Legendre equations |
| **Laplace Transforms** | Transform-based ODE solving, convolution, step/impulse functions |

## Key Symbols Used

| Symbol | Meaning |
|--------|---------|
| $y, y', y'', \ldots$ | Dependent variable and its derivatives |
| $x$ or $t$ | Independent variable |
| $p = dy/dx$ | First derivative shorthand |
| $D = d/dx$ | Differential operator |
| $W(y_1, y_2)$ | Wronskian determinant |
| $\mathcal{L}\{f(t)\}$ | Laplace transform of $f(t)$ |
| $\mathcal{L}^{-1}\{F(s)\}$ | Inverse Laplace transform |
| $u(t)$ | Unit step function |
| $\delta(t)$ | Dirac delta (impulse) function |
