# Calculus — Integral

## Course Overview

**Integral Calculus** is the branch of mathematics concerned with the theory and applications of integrals. While differential calculus deals with rates of change, integral calculus focuses on **accumulation** — areas, volumes, total quantities, and the reverse process of differentiation.

This course builds from the concept of antiderivatives through powerful integration techniques, and culminates in applications spanning geometry, physics, and engineering. By the end, you will be equipped to evaluate a wide variety of integrals and apply them to real-world problems.

### Core Themes

| Theme | Description |
|---|---|
| **Antidifferentiation** | Reversing the derivative to recover original functions |
| **Techniques** | Systematic methods (substitution, parts, partial fractions, trig) |
| **Definite Integrals** | Riemann sums, the Fundamental Theorem, and evaluation |
| **Applications** | Area, volume, arc length, surface area, physics problems |
| **Convergence** | Handling infinite limits and discontinuities rigorously |
| **Multiple Integrals** | Extending integration to functions of two variables |

### Prerequisites

- Differential Calculus (limits, derivatives, chain rule, product rule)
- Trigonometry (identities, inverse trig functions)
- Algebra (polynomial long division, partial fractions decomposition)

## Quick Reference — Essential Formulas

| Integral | Result |
|---|---|
| ∫ xⁿ dx | xⁿ⁺¹/(n+1) + C, n ≠ −1 |
| ∫ 1/x dx | ln\|x\| + C |
| ∫ eˣ dx | eˣ + C |
| ∫ aˣ dx | aˣ / ln a + C |
| ∫ sin x dx | −cos x + C |
| ∫ cos x dx | sin x + C |
| ∫ sec²x dx | tan x + C |
| ∫ csc²x dx | −cot x + C |
| ∫ sec x tan x dx | sec x + C |
| ∫ csc x cot x dx | −csc x + C |
| ∫ dx/√(1−x²) | sin⁻¹x + C |
| ∫ dx/(1+x²) | tan⁻¹x + C |
