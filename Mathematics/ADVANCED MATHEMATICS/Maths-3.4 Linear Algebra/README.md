## Course Overview

Linear Algebra is the branch of mathematics that studies **vectors, vector spaces, linear transformations, and systems of linear equations**. It provides the mathematical language and tools used extensively in engineering, physics, computer science, data science, machine learning, and many other fields.

These notes are organised into **8 units** that build upon each other:

| # | Unit | Core Idea |
|---|------|-----------|
| 1 | Vector Spaces | The abstract playground ‚Äî what vectors really are |
| 2 | Linear Transformations | Functions that respect vector structure |
| 3 | Inner Product Spaces | Adding geometry ‚Äî lengths, angles, projections |
| 4 | Eigenvalues & Eigenvectors | Directions preserved under a transformation |
| 5 | Diagonalization | Simplifying matrices via eigen-decomposition |
| 6 | Orthogonal Diagonalization | Exploiting symmetry ‚Äî the Spectral Theorem |
| 7 | SVD & Applications | The ultimate matrix factorisation |
| 8 | Abstract Vector Spaces | Beyond ‚Ñù‚Åø ‚Äî function spaces, polynomials, isomorphisms |
