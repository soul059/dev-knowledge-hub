## 2. Finding a Percentage of a Number

### The Formula
```

```

## 5. Quick Calculation Methods

### Finding 10% (The Building Block)

### Building Other Percentages from 10%
```

```

### Finding 50% (Halving)

## 6. Complex Percentage Calculations

### Finding Multiple Percentages

## 9. Solved Examples


### Example 2: Finding the Percentage

### Example 3: Finding the Whole
```

$9 Ã· 0.15 = $60

```

### Example 4: Multi-Step Problem
```
A store marks up items by 40%, then offers 25% off.
Original cost: $50. What's the final price?

Step 1: 40% markup
        40% of $50 = $20
        Marked price = $50 + $20 = $70

        25% of $70 = $17.50
        Final price = $70 - $17.50 = $52.50

Note: The final price ($52.50) is NOT the same as
      the original ($50). The order matters!
```
