## 1. What is a Percentage?

### Definition
**Percentage** means "per hundred" or "out of 100."

```
The word comes from Latin:
"per" = for each
"cent" = hundred

So 45% means 45 out of every 100
```

### The Symbol
```
% is the percentage symbol

25% = 25 per 100 = 25/100 = 0.25
```

## 7. Percentages Greater Than 100%

### Understanding Values Over 100%
```
100% = the whole (all of it)
More than 100% = more than the whole

If you have 150% of something, you have 1.5 times the original.
```

### Examples
```
Original price: $100

100% of $100 = $100 (same)
150% of $100 = $150 (1.5 times)
200% of $100 = $200 (double)
250% of $100 = $250 (2.5 times)
```

## 8. Percentages Less Than 1%

### Understanding Small Percentages
```
1% = 1/100 = 0.01
0.5% = 0.5/100 = 0.005
0.1% = 0.1/100 = 0.001
0.01% = 0.01/100 = 0.0001
```

### Real-World Uses
```
Interest rates: 0.5% annual rate
Statistics: 0.1% margin of error
Chemistry: 0.05% concentration
Medicine: 0.01% solution
```

### Conversion Examples
```
0.25% = 0.25/100 = 0.0025 (decimal)
      = 25/10000 = 1/400 (fraction)

0.75% = 0.75/100 = 0.0075 (decimal)
      = 75/10000 = 3/400 (fraction)
```

## 9. Comparing Using Percentages

### Making Comparisons
```
To compare different fractions, convert to percentages:

Which is larger: 3/5 or 5/8?

3/5 = 3 Ã· 5 = 0.6 = 60%
5/8 = 5 Ã· 8 = 0.625 = 62.5%

5/8 > 3/5 because 62.5% > 60%
```

### Test Score Example
```
Math test: 18/20
Science test: 42/50

Math: 18/20 = 0.9 = 90%
Science: 42/50 = 0.84 = 84%

Math score (90%) is better than Science (84%)
```
