### Formula
```

```

## 7. Finding Overall Percentage Change

### Formula for Two Changes
```
If change 1 = a% and change 2 = b%, then:

Overall multiplier = (1 Â± a/100) Ã— (1 Â± b/100)

Overall % change = (Overall multiplier - 1) Ã— 100
```

### Example Calculation
```
Find overall change: +30% then -20%

Multipliers: 1.30 Ã— 0.80 = 1.04

Overall change = (1.04 - 1) Ã— 100 = 4% increase
```

### Common Scenarios
```
+10% then -10%: 1.10 Ã— 0.90 = 0.99 â†’ 1% decrease
+20% then -20%: 1.20 Ã— 0.80 = 0.96 â†’ 4% decrease
```
