## 4. Comparing and Ordering Integers

### Rules for Comparing Integers

1. **Every positive integer > 0 > Every negative integer**
2. **Among positive integers**: Greater value = Greater number
3. **Among negative integers**: Greater magnitude = SMALLER number

### Examples
```
Compare the following:

1) 5 and -3
   5 > -3  âœ“ (positive > negative)

2) -7 and -2
   -7 < -2  âœ“ (further from 0 = smaller)
   
   On number line:
   â—â”â”â”â”â”â”â”â”â”â”â”â”â”â—â”â”â”â”â”â”â”â”â”â”â”â”â”â—
   -7           -2            0
   â†‘             â†‘
   Smaller      Larger

3) 0 and -5
   0 > -5  âœ“ (zero > any negative)

4) -100 and -1
   -100 < -1  âœ“ (even though 100 > 1)
```

### Ordering Integers (Ascending Order)
**Example**: Arrange -3, 5, -7, 0, 2, -1 in ascending order

**Step 1**: Identify negative and positive numbers
- Negative: -3, -7, -1
- Zero: 0
- Positive: 5, 2

**Step 2**: Order negatives (more negative = smaller)
- -7 < -3 < -1

**Step 3**: Add zero, then positives
- -7 < -3 < -1 < 0 < 2 < 5

**Answer**: -7, -3, -1, 0, 2, 5

## 6. Operations with Integers

### Addition of Integers

#### Rule 1: Same Signs â†’ ADD magnitudes, KEEP the sign
```
(+5) + (+3) = +8
(-5) + (-3) = -8
```

#### Rule 2: Different Signs â†’ SUBTRACT magnitudes, take sign of LARGER magnitude
```
(+7) + (-3) = +4   (7-3=4, positive wins)
(-7) + (+3) = -4   (7-3=4, negative wins)
(+3) + (-7) = -4   (7-3=4, negative wins)
```

### Subtraction of Integers

**Rule**: Subtracting is the same as adding the opposite (additive inverse)

```
a - b = a + (-b)
```

### Examples
```

(+8) - (-3) = (+8) + (+3) = +11
(-8) - (+3) = (-8) + (-3) = -11
(-8) - (-3) = (-8) + (+3) = -5
```
