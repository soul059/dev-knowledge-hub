## 1. What are Rational Numbers?

### Definition
A **rational number** is any number that can be expressed as a fraction p/q where:
- **p** is an integer (numerator)
- **q** is a non-zero integer (denominator)
- The symbol ‚Ñö represents rational numbers (from "Quotient")

```
‚Ñö = { p/q : p ‚àà ‚Ñ§, q ‚àà ‚Ñ§, q ‚â† 0 }
```

### Examples of Rational Numbers
```
Fractions:       1/2, 3/4, -5/7, 22/7
Integers:        5 = 5/1, -3 = -3/1, 0 = 0/1
Decimals:        0.5 = 1/2, 0.333... = 1/3
Mixed Numbers:   2¬Ω = 5/2
```

### Key Insight: All Integers are Rational Numbers
```
Every integer n can be written as n/1

Examples:
‚Ä¢ 7 = 7/1 ‚àà ‚Ñö
‚Ä¢ -15 = -15/1 ‚àà ‚Ñö
‚Ä¢ 0 = 0/1 ‚àà ‚Ñö (0 divided by any non-zero number is 0)
```

### Why q ‚â† 0?
```
Division by zero is UNDEFINED!

5/0 = ? (No answer exists)
‚Ä¢ If 5/0 = x, then 5 = 0 √ó x = 0 ‚úó (Contradiction!)

Therefore, denominator can NEVER be zero.
```

## 2. Standard Form of Rational Numbers

### Definition
A rational number is in **standard form** when:
1. The denominator is **positive**
2. The numerator and denominator have **no common factors** other than 1
3. The numerator and denominator are **coprime** (HCF = 1)

### Converting to Standard Form

**Step 1**: If denominator is negative, multiply both by -1
**Step 2**: Divide both numerator and denominator by their HCF

### Examples

**Example 1**: Convert -8/12 to standard form
```
Step 1: Denominator is positive (12 > 0) ‚úì
Step 2: Find HCF of 8 and 12
        Factors of 8: 1, 2, 4, 8
        Factors of 12: 1, 2, 3, 4, 6, 12
        HCF = 4

Step 3: Divide both by 4
        -8/12 = -8√∑4 / 12√∑4 = -2/3

Answer: -2/3 (Standard Form)
```

**Example 2**: Convert 15/-25 to standard form
```
Step 1: Denominator is negative, multiply by -1/-1
        15/-25 = (15 √ó -1) / (-25 √ó -1) = -15/25

Step 2: Find HCF of 15 and 25
        HCF = 5

Step 3: Divide both by 5
        -15/25 = -15√∑5 / 25√∑5 = -3/5

Answer: -3/5 (Standard Form)
```

**Example 3**: Convert -18/-24 to standard form
```
Step 1: Both negative, so positive result
        -18/-24 = 18/24 (or multiply by -1/-1)

Step 2: HCF of 18 and 24 = 6

Step 3: 18/24 = 18√∑6 / 24√∑6 = 3/4

Answer: 3/4 (Standard Form)
```

Two rational numbers are **equivalent** if they represent the same value.

```

```


## 5. Comparing Rational Numbers

### Method 1: Cross Multiplication

To compare a/b and c/d:
- If ad > bc, then a/b > c/d
- If ad < bc, then a/b < c/d
- If ad = bc, then a/b = c/d

### Example 1: Compare 3/4 and 5/7
```
Cross multiply:
3 √ó 7 = 21
4 √ó 5 = 20

Since 21 > 20,
3/4 > 5/7
```

### Method 2: Same Denominator

Convert to common denominator, then compare numerators.

### Example 2: Compare -5/6 and -7/8
```
LCM of 6 and 8 = 24

-5/6 = (-5 √ó 4)/(6 √ó 4) = -20/24
-7/8 = (-7 √ó 3)/(8 √ó 3) = -21/24

Since -20 > -21,
-5/6 > -7/8
```

### Method 3: Convert to Decimals

Divide numerator by denominator and compare decimal values.

### Example 3: Compare 2/5 and 3/8
```
2/5 = 0.4
3/8 = 0.375

Since 0.4 > 0.375,
2/5 > 3/8
```

### üí° Quick Tip for Negative Rationals
```
For negative rationals, the one with SMALLER absolute value is GREATER.

Compare -2/3 and -5/6:
|-2/3| = 2/3 ‚âà 0.667
|-5/6| = 5/6 ‚âà 0.833

Since 0.667 < 0.833,
-2/3 > -5/6 (less negative = greater)
```

## 6. Operations on Rational Numbers

### Addition and Subtraction

#### Same Denominator
```
a/c + b/c = (a + b)/c
a/c - b/c = (a - b)/c
```

**Example**: 3/7 + 2/7 = (3+2)/7 = 5/7

#### Different Denominators
**Step 1**: Find LCM of denominators
**Step 2**: Convert to equivalent fractions
**Step 3**: Add or subtract numerators

**Example**: 2/3 + 3/4
```
Step 1: LCM of 3 and 4 = 12

Step 2: 2/3 = (2√ó4)/(3√ó4) = 8/12
        3/4 = (3√ó3)/(4√ó3) = 9/12

Step 3: 8/12 + 9/12 = 17/12

Answer: 17/12 or 1 5/12
```

**Example**: 5/6 - 3/8
```
Step 1: LCM of 6 and 8 = 24

Step 2: 5/6 = (5√ó4)/(6√ó4) = 20/24
        3/8 = (3√ó3)/(8√ó3) = 9/24

Step 3: 20/24 - 9/24 = 11/24

Answer: 11/24
```

### Multiplication

**Rule**: Multiply numerators together, multiply denominators together
```
a/b √ó c/d = (a√óc)/(b√ód)
```

**Example**: 2/3 √ó 4/5
```
= (2√ó4)/(3√ó5)
= 8/15
```

**Example with simplification**: 3/4 √ó 8/9
```
Method 1 (Direct):
= (3√ó8)/(4√ó9) = 24/36 = 2/3

Method 2 (Cancel first - FASTER):
= 3/4 √ó 8/9
= (3√ó8)/(4√ó9)
  ‚Üì   ‚Üì
Cross-cancel: 3 and 9 have common factor 3
              4 and 8 have common factor 4

= (1√ó2)/(1√ó3) = 2/3
```

### Division

**Rule**: Multiply by the reciprocal of the divisor
```
a/b √∑ c/d = a/b √ó d/c = (a√ód)/(b√óc)
```

**Example**: 2/3 √∑ 4/5
```
= 2/3 √ó 5/4
= (2√ó5)/(3√ó4)
= 10/12
= 5/6 (simplified)
```

**Example**: -5/6 √∑ 10/9
```
= -5/6 √ó 9/10
= (-5√ó9)/(6√ó10)
= -45/60
= -3/4 (simplified)
```

## 7. Properties of Rational Numbers

### Summary Table

| Property | Addition | Multiplication |
|----------|----------|----------------|
| **Closure** | a/b + c/d ‚àà ‚Ñö | a/b √ó c/d ‚àà ‚Ñö |
| **Commutative** | a/b + c/d = c/d + a/b | a/b √ó c/d = c/d √ó a/b |
| **Associative** | (a/b + c/d) + e/f = a/b + (c/d + e/f) | (a/b √ó c/d) √ó e/f = a/b √ó (c/d √ó e/f) |
| **Identity** | a/b + 0 = a/b | a/b √ó 1 = a/b |
| **Inverse** | a/b + (-a/b) = 0 | a/b √ó b/a = 1 (if a‚â†0) |
| **Distributive** | a/b √ó (c/d + e/f) = a/b √ó c/d + a/b √ó e/f |

### Multiplicative Inverse (Reciprocal)

The **reciprocal** of a/b is b/a (where a ‚â† 0)

```
a/b √ó b/a = 1

Examples:
‚Ä¢ Reciprocal of 2/3 is 3/2
‚Ä¢ Reciprocal of -4/5 is -5/4 (or 5/-4)
‚Ä¢ Reciprocal of 7 (= 7/1) is 1/7
‚Ä¢ 0 has NO reciprocal (0/1 has no reciprocal 1/0)
```

## 8. Rational Numbers Between Two Numbers

### Key Concept
Between any two rational numbers, there are **infinitely many** rational numbers.

### Method 1: Average Method
To find a rational between a and b:
```
Rational number = (a + b) / 2
```

### Example: Find 3 rational numbers between 1/3 and 1/2
```
Step 1: Find one between 1/3 and 1/2
        Average = (1/3 + 1/2) √∑ 2
                = (2/6 + 3/6) √∑ 2
                = 5/6 √∑ 2
                = 5/12

Step 2: Find one between 1/3 and 5/12
        Average = (1/3 + 5/12) √∑ 2
                = (4/12 + 5/12) √∑ 2
                = 9/12 √∑ 2
                = 9/24 = 3/8

Step 3: Find one between 5/12 and 1/2
        Average = (5/12 + 1/2) √∑ 2
                = (5/12 + 6/12) √∑ 2
                = 11/12 √∑ 2
                = 11/24

Answer: 3/8, 5/12, 11/24 (all between 1/3 and 1/2)
```

### Method 2: Common Denominator Method
Convert to equivalent fractions with larger denominator.

### Example: Find 5 rational numbers between 3 and 4
```
Step 1: Write as fractions: 3/1 and 4/1
Step 2: Multiply both by 6/6 (to get more numbers between)
        3/1 = 18/6
        4/1 = 24/6
Step 3: Numbers between 18/6 and 24/6:
        19/6, 20/6, 21/6, 22/6, 23/6

Answer: 19/6, 10/3, 7/2, 11/3, 23/6
        (= 3‚Öô, 3‚Öì, 3¬Ω, 3‚Öî, 3‚Öö)
```

## 9. Solved Examples

### Example 1: Simplify (2/3 + 4/5) √∑ (1/2 - 1/3)

**Solution**:
```
Step 1: Solve the brackets
        Numerator: 2/3 + 4/5
        LCM of 3 and 5 = 15
        = 10/15 + 12/15 = 22/15

        Denominator: 1/2 - 1/3
        LCM of 2 and 3 = 6
        = 3/6 - 2/6 = 1/6

Step 2: Divide
        (22/15) √∑ (1/6)
        = 22/15 √ó 6/1
        = 132/15
        = 44/5

Answer: 44/5 or 8 4/5
```

### Example 2: The sum of two rational numbers is -5. If one of them is -13/6, find the other.

**Solution**:
```
Let the other number be x

x + (-13/6) = -5
x = -5 - (-13/6)
x = -5 + 13/6
x = -30/6 + 13/6
x = -17/6

Verification: -17/6 + (-13/6) = -30/6 = -5 ‚úì

Answer: -17/6
```

### Example 3: Find the product of additive inverse and multiplicative inverse of -2/3

**Solution**:
```
Additive inverse of -2/3 = 2/3  (sum = 0)
Multiplicative inverse of -2/3 = -3/2  (product = 1)

Product = 2/3 √ó (-3/2)
        = -6/6
        = -1

Answer: -1
```

### Example 4: Arrange in ascending order: 3/5, -2/3, 5/7, -1/2

**Solution**:
```
Step 1: LCM of 5, 3, 7, 2 = 210

Step 2: Convert each:
        3/5 = 126/210
        -2/3 = -140/210
        5/7 = 150/210
        -1/2 = -105/210

Step 3: Order numerators:
        -140 < -105 < 126 < 150

Step 4: Write original fractions:
        -2/3 < -1/2 < 3/5 < 5/7

Answer: -2/3, -1/2, 3/5, 5/7
```

### Example 5: If 2/5 of a number is 16, find 3/4 of the same number.

**Solution**:
```
Let the number be x

2/5 of x = 16
(2/5) √ó x = 16
x = 16 √ó 5/2
x = 80/2
x = 40

Now find 3/4 of 40:
3/4 √ó 40 = 120/4 = 30

Answer: 30
```
