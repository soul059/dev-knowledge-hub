## 7. Comparing Decimals

### Method: Compare Place by Place (Left to Right)
```
Step 1: Compare whole number parts
Step 2: If equal, compare tenths
Step 3: If equal, compare hundredths
Step 4: Continue until you find a difference
```

### Example 1: 3.45 vs 3.54
```
3.45 vs 3.54

Ones: 3 = 3 (equal)
Tenths: 4 < 5

Therefore: 3.45 < 3.54
```

### Example 2: 0.8 vs 0.75
```
0.8 vs 0.75

Write with same decimal places: 0.80 vs 0.75

Ones: 0 = 0 (equal)
Tenths: 8 > 7

Therefore: 0.8 > 0.75
```

### Example 3: 2.050 vs 2.05
```
2.050 vs 2.05

These are EQUAL!

Trailing zeros after decimal don't change value:
2.050 = 2.05 = 2.0500 = 2.05000
```

## 10. Equivalent Decimals

### Adding Zeros Doesn't Change Value
```
3 = 3.0 = 3.00 = 3.000 = 3.0000

0.5 = 0.50 = 0.500 = 0.5000

2.75 = 2.750 = 2.7500
```

### When to Add Zeros
```
For comparison: 0.5 â†’ 0.50 to compare with 0.45
For alignment: In column addition/subtraction
For precision: In scientific measurements
```

### Removing Unnecessary Zeros
```
5.200 â†’ 5.2 (trailing zeros)
0.040 â†’ 0.04 (trailing zeros)
007.5 â†’ 7.5 (leading zeros before decimal)

But keep: 0.040 â†’ 0.04 (zero before decimal stays for clarity)
```
