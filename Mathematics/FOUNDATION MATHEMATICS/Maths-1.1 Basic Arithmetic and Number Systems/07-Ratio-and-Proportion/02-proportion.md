## 4. Properties of Proportion

### Property 1: Invertendo
```
If a : b = c : d, then b : a = d : c

Example: If 2 : 3 = 4 : 6
Then: 3 : 2 = 6 : 4

(Invert both ratios)
```

### Property 2: Alternendo
```
If a : b = c : d, then a : c = b : d

Example: If 2 : 3 = 4 : 6
Then: 2 : 4 = 3 : 6 (which is 1 : 2 = 1 : 2)

(Switch the means)
```

### Property 3: Componendo
```
If a : b = c : d, then (a + b) : b = (c + d) : d

Example: If 3 : 4 = 6 : 8
Then: (3+4) : 4 = (6+8) : 8
      7 : 4 = 14 : 8 âœ“
```

### Property 4: Dividendo
```
If a : b = c : d, then (a - b) : b = (c - d) : d

Example: If 5 : 3 = 10 : 6
Then: (5-3) : 3 = (10-6) : 6
      2 : 3 = 4 : 6 âœ“
```

## 5. The Unitary Method

### Concept
```
The UNITARY METHOD finds the value of ONE unit first,
then uses it to find the required value.

Step 1: Find value of 1 unit
Step 2: Multiply to find required value
```

### Example 1: Direct Relationship
```
5 pens cost $15. What is the cost of 8 pens?

Step 1: Cost of 1 pen
        1 pen = $15 Ã· 5 = $3

Step 2: Cost of 8 pens
        8 pens = 8 Ã— $3 = $24
```

### Example 2: Working Backwards
```
If 12 workers can complete a job in 8 days,
how many days for 16 workers?

Step 1: Days for 1 worker
        1 worker = 12 Ã— 8 = 96 days

Step 2: Days for 16 workers
        16 workers = 96 Ã· 16 = 6 days

(Note: More workers = fewer days = inverse relationship)
```

## 7. Continued Proportion

### Definition
```
Quantities are in CONTINUED PROPORTION when:
a : b = b : c = c : d = ...

Each term (except first and last) is both a mean and an extreme.
```

### Example
```
2, 4, 8, 16 are in continued proportion:

2 : 4 = 4 : 8 = 8 : 16

All equal 1 : 2

This is a geometric sequence with common ratio 2.
```

### Finding Missing Terms
```

3 : x = x : 12

Check: 3 : 6 = 6 : 12 (both = 1 : 2) âœ“
```
