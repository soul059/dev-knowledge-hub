## 2. Solving Direct Proportion Problems

### Method 1: Using the Ratio
```
If 5 notebooks cost $20, how much do 8 notebooks cost?

Given: More notebooks â†’ More cost (direct proportion)

Set up ratio:
Notebooks : Cost = Notebooks : Cost
5 : 20 = 8 : x

5/20 = 8/x
5x = 20 Ã— 8
5x = 160
x = 32

8 notebooks cost $32
```

### Method 2: Finding the Constant
```
If 5 notebooks cost $20, how much do 8 notebooks cost?

Step 1: Find cost per notebook (constant)
        k = $20 Ã· 5 = $4 per notebook

Step 2: Use constant to find answer
        8 notebooks = 8 Ã— $4 = $32
```

### Method 3: Multiplier Method
```
If 5 notebooks cost $20, how much do 8 notebooks cost?

Multiplier: 8/5 = 1.6

Cost = $20 Ã— 1.6 = $32
```

## 6. The Constant of Proportionality

### In Direct Proportion
```
y = kx, where k is the constant

Example: Distance = Speed Ã— Time
If speed is constant at 60 km/h:
d = 60t

Here, k = 60 (the speed)
```

### In Inverse Proportion
```
xy = k, where k is the constant

Example: Workers Ã— Days = Total work
If total work is constant at 72 worker-days:
6 workers Ã— 12 days = 72
9 workers Ã— 8 days = 72

Here, k = 72 (total work)
```

### Finding the Constant
```
Direct: k = y/x (from any pair of values)
        If (3, 15) is a point, k = 15/3 = 5
        So y = 5x

Inverse: k = x Ã— y (from any pair of values)
         If (4, 6) is a point, k = 4 Ã— 6 = 24
         So xy = 24 or y = 24/x
```
