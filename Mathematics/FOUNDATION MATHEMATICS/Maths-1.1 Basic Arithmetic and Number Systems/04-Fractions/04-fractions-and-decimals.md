## 6. Comparing Fractions and Decimals

### Method 1: Convert Both to Decimals
```
Compare 3/8 and 0.4

3/8 = 0.375
0.4 = 0.400

0.375 < 0.400

Therefore: 3/8 < 0.4
```

### Method 2: Convert Both to Fractions
```
Compare 0.35 and 1/3

0.35 = 35/100 = 7/20
1/3 = ?/20 â†’ Need common denominator

LCM(20, 3) = 60
7/20 = 21/60
1/3 = 20/60

21/60 > 20/60

Therefore: 0.35 > 1/3
```

### Ordering Mixed Set
```
Order: 0.5, 1/3, 2/5, 0.45

Convert all to decimals:
0.5 = 0.5
1/3 = 0.333...
2/5 = 0.4
0.45 = 0.45

Order: 0.333... < 0.4 < 0.45 < 0.5
       1/3 < 2/5 < 0.45 < 0.5
```
