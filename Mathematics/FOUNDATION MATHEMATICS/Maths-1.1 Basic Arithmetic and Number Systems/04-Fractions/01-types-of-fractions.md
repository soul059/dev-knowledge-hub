## 6. Comparing Fractions

### Method 1: Same Denominator
```
When denominators are equal, compare numerators.

Compare 3/7 and 5/7:
Same denominator (7)
3 < 5
Therefore: 3/7 < 5/7

Visual:
```

### Method 2: Same Numerator
```
When numerators are equal, larger denominator = smaller fraction.

Compare 2/3 and 2/5:
Same numerator (2)
Larger denominator (5) = smaller pieces
Therefore: 2/5 < 2/3

Think: 2 slices of a pizza cut into 5 vs 3 pieces
       Smaller slices = less pizza!
```

### Method 3: Cross Multiplication
```
Compare a/b and c/d:
Cross multiply: a Ã— d vs c Ã— b

Compare 3/4 and 5/7:
3 Ã— 7 = 21
5 Ã— 4 = 20

21 > 20, so 3/4 > 5/7
```

### Method 4: Convert to Common Denominator
```
Compare 2/3 and 3/4:

LCM(3, 4) = 12

2/3 = 8/12
3/4 = 9/12

8/12 < 9/12
Therefore: 2/3 < 3/4
```

### Method 5: Convert to Decimals
```
Compare 5/8 and 7/11:

5/8 = 0.625
7/11 = 0.636...

0.625 < 0.636...
Therefore: 5/8 < 7/11
```

## 7. Ordering Fractions

### Ascending Order (Smallest to Largest)
```
Arrange 1/2, 2/3, 3/4, 5/6 in ascending order:

LCM(2, 3, 4, 6) = 12

1/2 = 6/12
2/3 = 8/12
3/4 = 9/12
5/6 = 10/12

Order: 6/12 < 8/12 < 9/12 < 10/12
       1/2 < 2/3 < 3/4 < 5/6
```

### Descending Order (Largest to Smallest)
```
Arrange 3/5, 2/3, 7/10, 1/2 in descending order:

LCM(5, 3, 10, 2) = 30

3/5 = 18/30
2/3 = 20/30
7/10 = 21/30
1/2 = 15/30

Order: 21/30 > 20/30 > 18/30 > 15/30
       7/10 > 2/3 > 3/5 > 1/2
```
