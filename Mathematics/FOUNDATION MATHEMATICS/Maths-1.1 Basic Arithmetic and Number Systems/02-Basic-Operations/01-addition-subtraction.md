## 4. Subtraction with Different Number Types

### Subtracting Integers

**Key Rule**: Subtracting is adding the opposite
```
a - b = a + (-b)
```

**Examples**:
```
(+8) - (+3) = (+8) + (-3) = +5
(+8) - (-3) = (+8) + (+3) = +11
(-8) - (+3) = (-8) + (-3) = -11
(-8) - (-3) = (-8) + (+3) = -5

## 5. Properties of Addition

### 1. Commutative Property
The order of addends doesn't affect the sum.
```
a + b = b + a

Examples:
5 + 3 = 3 + 5 = 8
(-7) + 4 = 4 + (-7) = -3
2.5 + 1.5 = 1.5 + 2.5 = 4
```

### 2. Associative Property
The grouping of addends doesn't affect the sum.
```
(a + b) + c = a + (b + c)

Example:
(2 + 3) + 4 = 2 + (3 + 4)
    5 + 4 = 2 + 7
        9 = 9 âœ“
```

### 3. Identity Property
Adding zero doesn't change the number.
```
a + 0 = a

Examples:
7 + 0 = 7
-5 + 0 = -5
3.14 + 0 = 3.14
```

### 4. Inverse Property
Every number has an opposite that sums to zero.
```
a + (-a) = 0

Examples:

## 7. Mental Math Strategies

### Strategy 1: Making Tens
```
8 + 7 = ?
Think: 8 + 2 = 10, then 10 + 5 = 15

8 + 7 = 8 + (2 + 5) = (8 + 2) + 5 = 10 + 5 = 15
```

### Strategy 2: Compensation
```
98 + 47 = ?
Think: 100 + 47 - 2 = 147 - 2 = 145

Or: 98 + 47 = 100 + 45 = 145
    (Add 2 to 98, subtract 2 from 47)
```

### Strategy 3: Breaking Apart
```
356 + 278 = ?

Break apart:
= 300 + 200 + 50 + 70 + 6 + 8
= 500 + 120 + 14
= 500 + 134
= 634
```

### Strategy 4: Counting Up (for Subtraction)
```
82 - 47 = ?

Count up from 47 to 82:
47 â†’ 50 (add 3)
50 â†’ 80 (add 30)
80 â†’ 82 (add 2)

Total added: 3 + 30 + 2 = 35
So 82 - 47 = 35
```

### Strategy 5: Same Change
```
83 - 28 = ?

Add same amount to both:
83 + 2 = 85
28 + 2 = 30

85 - 30 = 55
So 83 - 28 = 55
```

### Strategy 6: Near Doubles
```
7 + 8 = ?
Think: 7 + 7 = 14, then 14 + 1 = 15
Or:    8 + 8 = 16, then 16 - 1 = 15
```
