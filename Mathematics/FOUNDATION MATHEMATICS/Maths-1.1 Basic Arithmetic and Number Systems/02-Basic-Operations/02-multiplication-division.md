### Counting Negative Signs
```
For multiple factors, count the negative signs:
‚Ä¢ Even number of negatives ‚Üí Positive result
‚Ä¢ Odd number of negatives ‚Üí Negative result

(-2) √ó (-3) √ó (-4) = -24  (3 negatives, odd ‚Üí negative)
(-2) √ó (-3) √ó (-4) √ó (-1) = +24  (4 negatives, even ‚Üí positive)
(-1) √ó (-1) √ó (-1) √ó (-1) √ó (-1) = -1  (5 negatives, odd ‚Üí negative)
```

### Multiplying Fractions
```
Formula: (a/b) √ó (c/d) = (a√óc)/(b√ód)

Example: 2/3 √ó 4/5 = (2√ó4)/(3√ó5) = 8/15

With simplification (cross-cancel):
3/4 √ó 8/9 = (3√ó8)/(4√ó9)

Cancel 3 and 9: 3/9 = 1/3
Cancel 4 and 8: 8/4 = 2

= (1√ó2)/(1√ó3) = 2/3
```

### Multiplying Decimals
```
Step 1: Ignore decimals, multiply as whole numbers
Step 2: Count total decimal places in both factors
Step 3: Place decimal point in product

Example: 2.5 √ó 1.4
         25 √ó 14 = 350
         Total decimal places: 1 + 1 = 2
         Answer: 3.50 = 3.5

Example: 0.06 √ó 0.7
         6 √ó 7 = 42
         Total decimal places: 2 + 1 = 3
         Answer: 0.042
```

## 3. Understanding Division

### Definition
**Division** is the inverse of multiplication. It splits a number into equal parts or finds how many times one number fits into another.

### Terminology
```
    15   √∑    3    =    5
    ‚Üì         ‚Üì         ‚Üì
 Dividend   Divisor  Quotient

Also written as:
15/3 = 5  or  15 ‚îÄ 3 = 5  or  3)15
                                 5
```

### Division Concepts
```
15 √∑ 3 = 5 means:
‚Ä¢ "How many 3s fit into 15?" Answer: 5
‚Ä¢ "15 split into 3 equal parts gives?" Answer: 5 each
‚Ä¢ "What times 3 equals 15?" Answer: 5

‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè‚óè  √∑ 3 groups = ‚óè‚óè‚óè‚óè‚óè per group
      15                         5
```

### Relationship with Multiplication
```
If:     a √ó b = c
Then:   c √∑ b = a
And:    c √∑ a = b

Example:
6 √ó 4 = 24
24 √∑ 4 = 6
24 √∑ 6 = 4
```

## 4. Division with Different Number Types

### Dividing Integers

**Sign Rules (Same as Multiplication)**:
```
(+) √∑ (+) = (+)
(-) √∑ (-) = (+)
(+) √∑ (-) = (-)
(-) √∑ (+) = (-)

Examples:
(+20) √∑ (+4) = +5
(-20) √∑ (-4) = +5
(+20) √∑ (-4) = -5
(-20) √∑ (+4) = -5
```

### Division by Zero
```
‚ö†Ô∏è IMPORTANT: Division by zero is UNDEFINED!

5 √∑ 0 = ? (undefined - no answer exists)
0 √∑ 5 = 0 (this IS defined - zero divided by anything is zero)
0 √∑ 0 = ? (indeterminate - undefined)

Why is 5 √∑ 0 undefined?
If 5 √∑ 0 = x, then 0 √ó x = 5
But 0 √ó anything = 0, never 5!
So no answer exists.
```

### Dividing Fractions
```
Rule: Multiply by the reciprocal

(a/b) √∑ (c/d) = (a/b) √ó (d/c) = (a√ód)/(b√óc)

Example: 2/3 √∑ 4/5
       = 2/3 √ó 5/4
       = (2√ó5)/(3√ó4)
       = 10/12
       = 5/6

Memory trick: "Keep, Change, Flip"
Keep the first fraction
Change √∑ to √ó
Flip the second fraction
```

### Dividing Decimals
```
Rule: Move decimal points to make divisor a whole number

Example: 7.2 √∑ 0.4

Step 1: Move decimal in divisor to make it whole
        0.4 ‚Üí 4 (moved 1 place right)

Step 2: Move decimal in dividend same number of places
        7.2 ‚Üí 72 (moved 1 place right)

Step 3: Divide
        72 √∑ 4 = 18

Answer: 7.2 √∑ 0.4 = 18

Example: 15.75 √∑ 2.5
         157.5 √∑ 25 = 6.3
```

## 6. Properties of Multiplication

### 1. Commutative Property
Order doesn't matter.
```
a √ó b = b √ó a

Examples:
4 √ó 7 = 7 √ó 4 = 28
(-3) √ó 5 = 5 √ó (-3) = -15
```

### 2. Associative Property
Grouping doesn't matter.
```
(a √ó b) √ó c = a √ó (b √ó c)

Example:
(2 √ó 3) √ó 4 = 2 √ó (3 √ó 4)
    6 √ó 4 = 2 √ó 12
       24 = 24 ‚úì
```

### 3. Identity Property
Multiplying by 1 gives the same number.
```
a √ó 1 = a

Examples:
7 √ó 1 = 7
(-5) √ó 1 = -5
```

### 4. Zero Property
Multiplying by 0 gives 0.
```
a √ó 0 = 0

Examples:
999 √ó 0 = 0
(-1000) √ó 0 = 0
```

### 5. Distributive Property
Multiplication distributes over addition.
```
a √ó (b + c) = a √ó b + a √ó c

Example:
5 √ó (10 + 3) = 5 √ó 10 + 5 √ó 3
5 √ó 13 = 50 + 15
65 = 65 ‚úì

Also works with subtraction:
5 √ó (10 - 3) = 5 √ó 10 - 5 √ó 3
5 √ó 7 = 50 - 15
35 = 35 ‚úì
```

## 7. Properties of Division

### Division is NOT Commutative
```
a √∑ b ‚â† b √∑ a (in general)

Example:
12 √∑ 3 = 4
3 √∑ 12 = 0.25 = 1/4
4 ‚â† 0.25 ‚úó
```

### Division is NOT Associative
```
(a √∑ b) √∑ c ‚â† a √∑ (b √∑ c)

Example:
(24 √∑ 6) √∑ 2 = 4 √∑ 2 = 2
24 √∑ (6 √∑ 2) = 24 √∑ 3 = 8
2 ‚â† 8 ‚úó
```

### Division by 1
```
a √∑ 1 = a

Example: 15 √∑ 1 = 15
```

### Number Divided by Itself
```
a √∑ a = 1 (when a ‚â† 0)

Example: 7 √∑ 7 = 1
```

## 8. Mental Math Strategies

### Multiplication Tricks

**Multiplying by 5**:
```
Divide by 2, then multiply by 10
24 √ó 5 = 24 √∑ 2 √ó 10 = 12 √ó 10 = 120

Or: Multiply by 10, then divide by 2
24 √ó 5 = 240 √∑ 2 = 120
```

**Multiplying by 9**:
```
Multiply by 10, then subtract the original
23 √ó 9 = 23 √ó 10 - 23 = 230 - 23 = 207
```

**Multiplying by 11**:
```
For 2-digit numbers: Add the digits, put in middle
35 √ó 11 = 3_(3+5)_5 = 385

If sum > 9, carry over:
78 √ó 11 = 7_(7+8)_8 = 7_(15)_8 = 858
```

**Multiplying by 25**:
```
Divide by 4, then multiply by 100
36 √ó 25 = 36 √∑ 4 √ó 100 = 9 √ó 100 = 900
```

**Squaring Numbers Ending in 5**:
```
For n5 √ó n5:
‚Ä¢ Multiply n by (n+1)
‚Ä¢ Append 25

25¬≤ = (2 √ó 3) followed by 25 = 625
35¬≤ = (3 √ó 4) followed by 25 = 1225
65¬≤ = (6 √ó 7) followed by 25 = 4225
```

### Division Tricks

**Dividing by 5**:
```
Multiply by 2, then divide by 10
135 √∑ 5 = 135 √ó 2 √∑ 10 = 270 √∑ 10 = 27
```

**Dividing by 25**:
```
Multiply by 4, then divide by 100
350 √∑ 25 = 350 √ó 4 √∑ 100 = 1400 √∑ 100 = 14
```

**Dividing by 4**:
```
Halve it, then halve again
96 √∑ 4 = 96 √∑ 2 √∑ 2 = 48 √∑ 2 = 24
```

## 10. Common Mistakes to Avoid

### Mistake 1: Sign Errors
```
Wrong: (-3) √ó (-4) = -12 ‚úó
Correct: (-3) √ó (-4) = +12 ‚úì (same signs = positive)

Wrong: (-8) √∑ (+2) = +4 ‚úó
Correct: (-8) √∑ (+2) = -4 ‚úì (different signs = negative)
```

### Mistake 2: Decimal Point Errors
```
Wrong: 2.5 √ó 0.3 = 7.5 ‚úó
Correct: 2.5 √ó 0.3 = 0.75 ‚úì
(25 √ó 3 = 75, then 2 decimal places ‚Üí 0.75)
```

### Mistake 3: Dividing Fractions
```
Wrong: (2/3) √∑ (4/5) = (2√ó4)/(3√ó5) = 8/15 ‚úó
Correct: (2/3) √∑ (4/5) = (2/3) √ó (5/4) = 10/12 = 5/6 ‚úì
```

### Mistake 4: Order in Division
```
Wrong: "12 divided by 3" = 3/12 ‚úó
Correct: "12 divided by 3" = 12/3 = 4 ‚úì
```
