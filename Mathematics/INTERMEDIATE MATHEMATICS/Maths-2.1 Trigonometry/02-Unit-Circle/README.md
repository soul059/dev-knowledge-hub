# Unit 2: Unit Circle

## Overview

The unit circle is a powerful tool that extends trigonometric ratios beyond acute angles to any real number. This unit introduces the unit circle, explains how trigonometric functions behave in different quadrants, and shows how to find ratios for any angle.
