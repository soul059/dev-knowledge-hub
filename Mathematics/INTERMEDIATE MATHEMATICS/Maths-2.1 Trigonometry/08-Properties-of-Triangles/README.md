### Convention

- **Vertices**: Capital letters A, B, C
- **Angles**: At vertices A, B, C (also denoted A, B, C)
- **Sides**: Lowercase a, b, c opposite to angles A, B, C respectively
  - Side a (BC) is opposite to angle A
  - Side b (CA) is opposite to angle B
  - Side c (AB) is opposite to angle C

## ğŸ”‘ Key Formulas Preview

### Sine Rule

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

### Cosine Rule

$$a^2 = b^2 + c^2 - 2bc\cos A$$

### Area Formulas

$$\Delta = \frac{1}{2}bc\sin A = \sqrt{s(s-a)(s-b)(s-c)}$$

### Circumradius (R)

$$R = \frac{abc}{4\Delta}$$

### Inradius (r)

$$r = \frac{\Delta}{s}$$

where $s = \frac{a+b+c}{2}$ (semi-perimeter)
