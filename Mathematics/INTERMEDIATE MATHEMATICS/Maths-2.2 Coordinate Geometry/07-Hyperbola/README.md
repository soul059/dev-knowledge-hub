# Unit 7: Hyperbola

## ðŸ“š Unit Overview

The **hyperbola** is the third major conic section, formed when a plane intersects both nappes of a double cone. Unlike ellipse where the sum of focal distances is constant, in hyperbola the **difference** of focal distances is constant. This unit explores the hyperbola's equation, properties, and applications.

## ðŸ”‘ Key Formulas

### Standard Equations

| Form | Equation | Transverse Axis |
|------|----------|-----------------|
| Horizontal | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ | Along x-axis |
| Vertical | $\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1$ | Along y-axis |

### Key Relations

| Property | Formula |
|----------|---------|
| Relation | $c^2 = a^2 + b^2$ |
| Eccentricity | $e = \frac{c}{a} > 1$ |
| Directrix | $x = \pm\frac{a}{e}$ |
| Latus Rectum | $\ell = \frac{2b^2}{a}$ |
| Focal distance | $|SP - S'P| = 2a$ |
| Asymptotes | $y = \pm\frac{b}{a}x$ |

## ðŸ”„ Comparison: Ellipse vs Hyperbola

| Property | Ellipse | Hyperbola |
|----------|---------|-----------|
| Equation | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ |
| Relation | $c^2 = a^2 - b^2$ | $c^2 = a^2 + b^2$ |
| Eccentricity | $0 < e < 1$ | $e > 1$ |
| Focal sum/diff | $SP + S'P = 2a$ | $|SP - S'P| = 2a$ |
| Shape | Closed curve | Two open branches |
| Asymptotes | None | $y = \pm\frac{b}{a}x$ |

## âš¡ Quick Reference

### Parametric Form
| Ellipse | Hyperbola |
|---------|-----------|
| $(a\cos\theta, b\sin\theta)$ | $(a\sec\theta, b\tan\theta)$ |

### Tangent at Point $(x_1, y_1)$
| Ellipse | Hyperbola |
|---------|-----------|
| $\frac{xx_1}{a^2} + \frac{yy_1}{b^2} = 1$ | $\frac{xx_1}{a^2} - \frac{yy_1}{b^2} = 1$ |

### Condition for Line $y = mx + c$ to be Tangent
| Ellipse | Hyperbola |
|---------|-----------|
| $c^2 = a^2m^2 + b^2$ | $c^2 = a^2m^2 - b^2$ |
