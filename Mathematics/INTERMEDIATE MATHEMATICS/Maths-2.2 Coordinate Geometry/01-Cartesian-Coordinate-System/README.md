## âš¡ Key Formulas Summary

| Concept | Formula |
|---------|---------|
| Distance between $(x_1, y_1)$ and $(x_2, y_2)$ | $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ |
| Internal Section $(m:n)$ | $\left(\frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n}\right)$ |
| External Section $(m:n)$ | $\left(\frac{mx_2-nx_1}{m-n}, \frac{my_2-ny_1}{m-n}\right)$ |
| Midpoint | $\left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)$ |
| Area of Triangle | $\frac{1}{2}\|x_1(y_2-y_3) + x_2(y_3-y_1) + x_3(y_1-y_2)\|$ |
| Collinearity Condition | Area = 0 |
