## Course Overview

**Coordinate Geometry** (also known as **Analytic Geometry**) is the study of geometry using a coordinate system. This powerful branch of mathematics bridges algebra and geometry, allowing us to represent geometric shapes using algebraic equations and solve geometric problems using algebraic techniques.

### Why Study Coordinate Geometry?

- **Foundation for Calculus**: Essential prerequisite for differential and integral calculus
- **Engineering Applications**: Used in CAD, computer graphics, robotics, and navigation
- **Physics**: Describes motion, forces, and trajectories
- **Real-world Problem Solving**: Maps, GPS, architecture, and game development

### Prerequisites

- Basic algebra and equations
- Fundamentals of Euclidean geometry
- Trigonometry basics
- Understanding of functions

## ğŸ”¢ Quick Formula Reference

### Basic Formulas

| Formula | Expression |
|---------|------------|
| Distance | $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ |
| Midpoint | $M = \left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)$ |
| Section (Internal) | $P = \left(\frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n}\right)$ |
| Slope | $m = \frac{y_2-y_1}{x_2-x_1} = \tan\theta$ |
| Area of Triangle | $A = \frac{1}{2}|x_1(y_2-y_3) + x_2(y_3-y_1) + x_3(y_1-y_2)|$ |

### Line Equations

| Form | Equation |
|------|----------|
| Slope-Intercept | $y = mx + c$ |
| Point-Slope | $y - y_1 = m(x - x_1)$ |
| Two-Point | $\frac{y-y_1}{y_2-y_1} = \frac{x-x_1}{x_2-x_1}$ |
| Intercept | $\frac{x}{a} + \frac{y}{b} = 1$ |
| Normal | $x\cos\alpha + y\sin\alpha = p$ |
| General | $ax + by + c = 0$ |

### Conic Sections

| Curve | Standard Equation | Eccentricity |
|-------|-------------------|--------------|
| Circle | $x^2 + y^2 = r^2$ | $e = 0$ |
| Parabola | $y^2 = 4ax$ | $e = 1$ |
| Ellipse | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ | $0 < e < 1$ |
| Hyperbola | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ | $e > 1$ |
