## âš¡ Quick Reference: Standard Forms

| Parabola | Equation | Focus | Directrix | Axis | Opens |
|----------|----------|-------|-----------|------|-------|
| Right | $y^2 = 4ax$ | $(a, 0)$ | $x = -a$ | y = 0 | Right |
| Left | $y^2 = -4ax$ | $(-a, 0)$ | $x = a$ | y = 0 | Left |
| Up | $x^2 = 4ay$ | $(0, a)$ | $y = -a$ | x = 0 | Up |
| Down | $x^2 = -4ay$ | $(0, -a)$ | $y = a$ | x = 0 | Down |

## ğŸ“ Key Formulas Summary

| Property | Formula/Value |
|----------|---------------|
| Definition | Distance from focus = Distance from directrix |
| Parametric point | $(at^2, 2at)$ for $y^2 = 4ax$ |
| Latus rectum length | $4a$ |
| Focal distance | $a + x_1$ for point $(x_1, y_1)$ |
| Tangent at $(x_1, y_1)$ | $yy_1 = 2a(x + x_1)$ |
| Tangent at parameter $t$ | $ty = x + at^2$ |
| Tangent with slope $m$ | $y = mx + \frac{a}{m}$ |
| Normal at $(x_1, y_1)$ | $y - y_1 = -\frac{y_1}{2a}(x - x_1)$ |
| Normal at parameter $t$ | $y + tx = 2at + at^3$ |
