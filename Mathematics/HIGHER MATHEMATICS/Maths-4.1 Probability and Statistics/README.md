## Course Overview

**Probability and Statistics** is a fundamental branch of mathematics that deals with the analysis of random phenomena, data interpretation, and decision-making under uncertainty. This course builds from basic probability axioms through distributions, estimation, hypothesis testing, and regression analysis.

### What You'll Learn

| Area | Topics Covered |
|------|---------------|
| **Foundations** | Sample spaces, events, counting principles, probability axioms |
| **Random Variables** | Discrete & continuous variables, PMF, PDF, CDF |
| **Distributions** | Bernoulli, Binomial, Poisson, Normal, Exponential, Chi-square, t, F |
| **Descriptive Measures** | Expectation, variance, moments, covariance, correlation |
| **Joint Analysis** | Joint, marginal, conditional distributions |
| **Inference** | Sampling, estimation, confidence intervals |
| **Testing** | Hypothesis tests, Z-test, t-test, Chi-square, ANOVA |
| **Modeling** | Linear regression, least squares, multiple regression |

### Prerequisites

- Calculus (differentiation and integration)
- Basic set theory
- Linear algebra fundamentals (for regression)

## Key Symbols and Notation

| Symbol | Meaning |
|--------|---------|
| $S$ or $\Omega$ | Sample space |
| $P(A)$ | Probability of event A |
| $P(A \mid B)$ | Conditional probability of A given B |
| $X, Y, Z$ | Random variables |
| $f(x)$ | PDF (continuous) or PMF (discrete) |
| $F(x)$ | CDF |
| $E[X]$ or $\mu$ | Expected value / mean |
| $\text{Var}(X)$ or $\sigma^2$ | Variance |
| $\sigma$ | Standard deviation |
| $\binom{n}{r}$ | Binomial coefficient "n choose r" |
| $n!$ | n factorial |
| $\sum$ | Summation |
| $\int$ | Integration |
| $\bar{X}$ | Sample mean |
| $s^2$ | Sample variance |
| $H_0, H_1$ | Null and alternative hypothesis |
| $\alpha$ | Significance level |
| $\beta$ | Type II error probability |
| $\rho$ | Population correlation coefficient |
| $r$ | Sample correlation coefficient |
