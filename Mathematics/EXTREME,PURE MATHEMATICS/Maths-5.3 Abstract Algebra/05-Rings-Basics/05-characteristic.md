## Overview

The **characteristic** of a ring measures the "additive order" of $1$. It determines whether the ring "feels like" characteristic 0 (like $\mathbb{Z}$) or characteristic $p$ (like $\mathbb{F}_p$). In characteristic $p$, the Frobenius endomorphism has remarkable properties.

---

## 1. Definition

**Definition.** The **characteristic** of a ring $R$ with unity is:

$$\text{char}(R) = \begin{cases} \min\{n \in \mathbb{Z}^+ : n \cdot 1_R = 0\} & \text{if such } n \text{ exists} \\ 0 & \text{otherwise} \end{cases}$$

where $n \cdot 1_R = \underbrace{1_R + 1_R + \cdots + 1_R}_{n}$.

**Equivalently:** $\text{char}(R)$ is the non-negative generator of $\ker(\phi)$, where $\phi: \mathbb{Z} \to R$ is the unique ring homomorphism $n \mapsto n \cdot 1_R$.

```
  Unique ring homomorphism ℤ → R:
  
  φ: n ↦ n · 1_R
  
  ker(φ) = nℤ  ⟹  char(R) = n
  
  char(R) = 0  ⟺  ker(φ) = {0}  ⟺  ℤ ↪ R
  char(R) = n  ⟺  ker(φ) = nℤ   ⟺  ℤ/nℤ ↪ R
```

---

## 2. Examples

| Ring $R$ | $\text{char}(R)$ | Reason |
|----------|-----------------|--------|
| $\mathbb{Z}$ | 0 | No positive $n$ with $n \cdot 1 = 0$ |
| $\mathbb{Q}, \mathbb{R}, \mathbb{C}$ | 0 | Contain $\mathbb{Z}$ |
| $\mathbb{Z}_n$ | $n$ | $n \cdot \bar{1} = \bar{0}$ |
| $\mathbb{F}_p$ | $p$ | $p \cdot 1 = 0$ |
| $\mathbb{F}_{p^n}$ | $p$ | Contains $\mathbb{F}_p$ |
| $M_n(\mathbb{F}_p)$ | $p$ | $p \cdot I_n = 0$ |
| $\mathbb{Z}[x]$ | 0 | Contains $\mathbb{Z}$ |
| $\mathbb{F}_p[x]$ | $p$ | Contains $\mathbb{F}_p$ |
| $\mathbb{Z} \times \mathbb{Z}_3$ | 0 | $(n, 0) \neq (0,0)$ for any $n > 0$ unless... actually $1 = (1,1)$, $n \cdot(1,1) = (n, n \bmod 3)$; need both $= 0$, so $\text{char} = 0$. |

## 3. Key Theorems

### Theorem 5.15: Characteristic of Integral Domains

**Theorem.** If $R$ is an integral domain, then $\text{char}(R) = 0$ or $\text{char}(R) = p$ (prime).

*Proof.* If $\text{char}(R) = n = ab$ with $1 < a, b < n$:
$0 = n \cdot 1 = (a \cdot 1)(b \cdot 1)$. Since $R$ is a domain, $a \cdot 1 = 0$ or $b \cdot 1 = 0$, contradicting minimality. $\blacksquare$

### Theorem 5.16: Characteristic and Prime Subring

The image of $\phi: \mathbb{Z} \to R$ is the **prime subring** of $R$:

$$\text{Im}(\phi) \cong \begin{cases} \mathbb{Z} & \text{if } \text{char}(R) = 0 \\ \mathbb{Z}_n & \text{if } \text{char}(R) = n \end{cases}$$

For an integral domain, the **prime subfield** is:

$$\begin{cases} \mathbb{Q} & \text{if } \text{char}(R) = 0 \\ \mathbb{F}_p & \text{if } \text{char}(R) = p \end{cases}$$

## 4. The Frobenius Endomorphism

**Theorem 5.17 (Freshman's Dream).** In a commutative ring of characteristic $p$:

$$(a + b)^p = a^p + b^p$$

*Proof.* By the binomial theorem:

$$(a + b)^p = \sum_{k=0}^{p} \binom{p}{k} a^k b^{p-k}$$

For $0 < k < p$: $\binom{p}{k} = \frac{p!}{k!(p-k)!}$. Since $p$ is prime and $p \mid p!$ but $p \nmid k!(p-k)!$ (both factors are $< p$), we have $p \mid \binom{p}{k}$. In characteristic $p$, this term vanishes. $\blacksquare$

**Definition.** The **Frobenius endomorphism** of a commutative ring $R$ with $\text{char}(R) = p$ is:

$$\sigma: R \to R, \quad \sigma(a) = a^p$$

By the Freshman's Dream: $\sigma(a + b) = \sigma(a) + \sigma(b)$, and clearly $\sigma(ab) = \sigma(a)\sigma(b)$, $\sigma(1) = 1$.

```
  Frobenius map:  σ(a) = aᵖ
  
  Properties:
  • σ is a ring homomorphism
  • σ is always injective (in a domain)
  • σ is surjective ⟺ "perfect" field
  • For finite fields: σ is an automorphism
  
  Example in F₃:  σ(0)=0, σ(1)=1, σ(2)=2³=8=2
  So σ = id on F₃.
  
  Example in F₄ = {0,1,α,α+1}:
  σ(α) = α² = α+1
  σ(α+1) = (α+1)² = α² + 1 = α  (char 2!)
  So σ swaps α and α+1.  σ² = id.
```

## 5. Iterated Frobenius

In $\text{char} = p$:

$$(a + b)^{p^n} = a^{p^n} + b^{p^n}$$

The $n$-th Frobenius is $\sigma^n: a \mapsto a^{p^n}$.

For a finite field $\mathbb{F}_{p^n}$: the Frobenius $\sigma$ has order $n$, i.e., $\sigma^n = \text{id}$ but $\sigma^k \neq \text{id}$ for $0 < k < n$.

The Galois group $\text{Gal}(\mathbb{F}_{p^n}/\mathbb{F}_p)$ is cyclic of order $n$, generated by $\sigma$.

---

## 6. Characteristic and Tensor Products

For a field $F$ with $\text{char}(F) = p$:

- Derivatives in $F[x]$ can behave strangely: $\frac{d}{dx}(x^p) = px^{p-1} = 0$.
- A polynomial can have repeated roots even when its formal derivative is zero.
- This affects separability: $f(x)$ is **inseparable** if $f'(x) = 0$ (only possible in char $p$).

```
  Char p pathologies:
  
  • (a + b)ᵖ = aᵖ + bᵖ     ("freshman's dream")
  • d/dx (xᵖ) = 0           (derivative vanishes)
  • xᵖ - a may be irreducible but inseparable
  • Not all extensions are separable
```

## 7. Characteristic in Practice

| Property | $\text{char} = 0$ | $\text{char} = p$ |
|----------|-------------------|---------------------|
| $n \cdot 1 = 0$ iff | $n = 0$ | $p \mid n$ |
| $-1 = 1$ | Never | Iff $p = 2$ |
| Binomial theorem | Full version | $(a+b)^p = a^p + b^p$ |
| Frobenius | N/A | $a \mapsto a^p$ is a homomorphism |
| Prime subfield | $\mathbb{Q}$ | $\mathbb{F}_p$ |
| Derivative of $x^p$ | $px^{p-1} \neq 0$ | $0$ |

---

## 8. Characteristic 2: Special Features

In characteristic 2:
- $-1 = 1$ (so $a = -a$ for all $a$)
- $a + a = 0$ for all $a$ (every element has order 2 in $(R, +)$)
- $(a + b)^2 = a^2 + b^2$ (no cross terms)
- $\mathbb{F}_2 = \{0, 1\}$: the smallest field

Boolean algebra naturally lives in characteristic 2.
