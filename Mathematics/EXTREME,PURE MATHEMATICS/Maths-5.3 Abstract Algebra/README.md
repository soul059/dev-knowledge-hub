# Abstract Algebra — Comprehensive Study Notes

> **A complete course covering Groups, Rings, and Fields with rigorous theory, structural diagrams, worked examples, and revision aids.**

---

## Course Overview

Abstract Algebra (also called Modern Algebra) is the study of algebraic structures — sets equipped with operations that satisfy certain axioms. This course systematically builds from the basic notion of a **group** through **rings** to **fields**, culminating in an introduction to **Galois theory**.

### What You Will Learn

| Area | Key Topics | Core Skill |
|------|-----------|------------|
| **Groups** | Axioms, subgroups, homomorphisms, Sylow theorems | Classifying & analysing symmetries |
| **Rings** | Ideals, quotient rings, factorisation, PIDs, UFDs | Generalising number theory |
| **Fields** | Extensions, finite fields, splitting fields, Galois theory | Solving polynomial equations |

### Prerequisites

- Familiarity with sets, functions, and relations
- Basic number theory (divisibility, primes, modular arithmetic)
- Mathematical maturity (comfort with proofs)

### How to Use These Notes

1. **Read sequentially** — each chapter builds on previous ones.
2. **Work through the examples** — algebra is learned by doing.
3. **Use the summary tables** for quick review before exams.
4. **Attempt the revision questions** at the end of each chapter.

## Structural Hierarchy of Algebra

```
                    ┌─────────────────────────┐
                    │       MAGMA (S, ∗)       │  Closed binary operation
                    └────────────┬────────────┘
                                 │ + Associativity
                    ┌────────────▼────────────┐
                    │      SEMIGROUP           │
                    └────────────┬────────────┘
                                 │ + Identity
                    ┌────────────▼────────────┐
                    │       MONOID             │
                    └────────────┬────────────┘
                                 │ + Inverses
                    ┌────────────▼────────────┐
                    │        GROUP             │
                    └────────────┬────────────┘
                                 │ + Commutativity
                    ┌────────────▼────────────┐
                    │    ABELIAN GROUP         │
                    └────────────┬────────────┘
                                 │ + Second operation (multiplication)
                    ┌────────────▼────────────┐
                    │         RING             │
                    └───┬────────────────┬─────┘
                        │                │
          + No zero     │                │ + Commutativity of ×
            divisors    │                │
          ┌─────────────▼──┐   ┌────────▼──────────┐
          │INTEGRAL DOMAIN │   │ COMMUTATIVE RING   │
          └───────┬────────┘   └────────────────────┘
                  │ + Every nonzero element invertible
          ┌───────▼────────┐
          │     FIELD       │
          └────────────────┘
```

## Quick Reference: Key Symbols

| Symbol | Meaning |
|--------|---------|
| $(G, \ast)$ | Group $G$ with operation $\ast$ |
| $e$ or $1_G$ | Identity element |
| $a^{-1}$ | Inverse of $a$ |
| $\|G\|$ or $\text{ord}(G)$ | Order of group $G$ |
| $\|a\|$ or $\text{ord}(a)$ | Order of element $a$ |
| $H \leq G$ | $H$ is a subgroup of $G$ |
| $H \trianglelefteq G$ | $H$ is a normal subgroup of $G$ |
| $G/H$ | Quotient group |
| $\phi: G \to H$ | Homomorphism |
| $\ker \phi$ | Kernel of $\phi$ |
| $\text{im}\,\phi$ | Image of $\phi$ |
| $G \cong H$ | $G$ is isomorphic to $H$ |
| $\langle a \rangle$ | Cyclic group generated by $a$ |
| $(R, +, \cdot)$ | Ring |
| $I \trianglelefteq R$ | $I$ is an ideal of $R$ |
| $R/I$ | Quotient ring |
| $F \subseteq E$ | $F$ is a subfield of $E$ |
| $[E : F]$ | Degree of field extension |
