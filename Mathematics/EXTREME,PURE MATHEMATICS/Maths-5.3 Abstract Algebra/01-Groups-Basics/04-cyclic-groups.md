# Chapter 1.4 — Cyclic Groups

## Overview

Cyclic groups are the simplest and most well-understood class of groups. Every cyclic group is generated by a single element. Despite their simplicity, they appear everywhere — modular arithmetic, roots of unity, and as building blocks of all finitely generated abelian groups.

---

## 1. Definition

**Definition.** A group $G$ is **cyclic** if there exists $a \in G$ such that:

$$G = \langle a \rangle = \{a^n : n \in \mathbb{Z}\}$$

The element $a$ is called a **generator** of $G$.

### In Additive Notation

$$G = \langle a \rangle = \{na : n \in \mathbb{Z}\}$$

---

## 2. The Two Types of Cyclic Groups

**Theorem 1.16 (Fundamental theorem of cyclic groups).** Every cyclic group is isomorphic to either $\mathbb{Z}$ or $\mathbb{Z}_n$ for some $n \geq 1$.

```
  Cyclic Groups
  ┌─────────────────────────────────────────────────┐
  │                                                  │
  │   Infinite cyclic           Finite cyclic        │
  │   ⟨a⟩ ≅ Z                  ⟨a⟩ ≅ Zₙ             │
  │   |a| = ∞                  |a| = n               │
  │                                                  │
  │   Elements:                Elements:              │
  │   ..., a⁻², a⁻¹, e,       e, a, a², ..., aⁿ⁻¹  │
  │   a, a², ...               (aⁿ = e)              │
  │                                                  │
  │   ← ──── Number line ────→  ● ─── Circle        │
  │   ...−2 −1  0  1  2  ...                         │
  └─────────────────────────────────────────────────┘
```

**Visualising $\mathbb{Z}_n$ as clock arithmetic:**

```
                    0
                   / \
                 11    1
                /        \
              10           2
              |             |         Z₁₂ ≅ ⟨1⟩
               9           3          
                \         /
                  8     4
                   \  /
                 7 ─ 6 ─ 5
```

## 3. Subgroups of Cyclic Groups

### Theorem 1.17: Every Subgroup of a Cyclic Group is Cyclic

If $G = \langle a \rangle$ and $H \leq G$, then $H = \langle a^d \rangle$ for some $d \geq 0$.

### Theorem 1.18: Subgroups of $\mathbb{Z}_n$

For each divisor $d$ of $n$, there is **exactly one** subgroup of $\mathbb{Z}_n$ of order $d$, namely $\langle n/d \rangle$.

**Conversely,** every subgroup of $\mathbb{Z}_n$ has order dividing $n$.

**Example — Subgroup lattice of $\mathbb{Z}_{24}$:**

Divisors of 24: $1, 2, 3, 4, 6, 8, 12, 24$

```
                         Z₂₄ = ⟨1⟩              order 24
                       /   |   \
                     /     |     \
                  ⟨2⟩     ⟨3⟩           order 12, 8
                 / | \     |   \
               /   |  \    |    \
            ⟨4⟩   ⟨6⟩  ⟨3⟩  ⟨8⟩         order 6, 4, 3
             | \ / |    |
             | /\  |    |
           ⟨8⟩ ⟨12⟩     |               order 3, 2
              \  |    /
               \ |  /
                ⟨12⟩                     order 2
                 |
                {0}                      order 1

  Corrected:
  
           ⟨1⟩ = Z₂₄       d=24
          /    |    \
       ⟨2⟩   ⟨3⟩         d=12,  d=8
       / \   / \
    ⟨4⟩  ⟨6⟩  ⟨3⟩         d=6, d=4, (d=8 connects to d=4)
     |  X   |
    ⟨8⟩ ⟨12⟩               d=3, d=2
      \  /
      {0}                  d=1
```

### Counting Subgroups

The number of subgroups of $\mathbb{Z}_n$ equals the number of divisors of $n$, denoted $\tau(n)$.

| $n$ | Divisors | $\tau(n)$ |
|-----|----------|-----------|
| 6 | 1, 2, 3, 6 | 4 |
| 12 | 1, 2, 3, 4, 6, 12 | 6 |
| $p$ (prime) | 1, $p$ | 2 |
| $p^k$ | $1, p, p^2, \ldots, p^k$ | $k+1$ |

## 4. Generators of $\mathbb{Z}_n$

**Theorem 1.19.** $a$ is a generator of $\mathbb{Z}_n$ if and only if $\gcd(a, n) = 1$.

The number of generators of $\mathbb{Z}_n$ is $\varphi(n)$ (Euler's totient).

**Example — Generators of $\mathbb{Z}_{12}$:**

| Element | $\gcd(a, 12)$ | Generator? |
|---------|---------------|-----------|
| 0 | 12 | No |
| 1 | 1 | **Yes** |
| 2 | 2 | No |
| 3 | 3 | No |
| 4 | 4 | No |
| 5 | 1 | **Yes** |
| 6 | 6 | No |
| 7 | 1 | **Yes** |
| 8 | 4 | No |
| 9 | 3 | No |
| 10 | 2 | No |
| 11 | 1 | **Yes** |

Generators: $\{1, 5, 7, 11\}$, count $= \varphi(12) = 4$. ✓

## 5. Properties of Cyclic Groups

### Theorem 1.20: Cyclic Groups are Abelian

Every cyclic group is abelian.

*Proof.* $a^m \cdot a^n = a^{m+n} = a^{n+m} = a^n \cdot a^m$. $\square$

> **Warning:** The converse is false. $V_4 = \mathbb{Z}_2 \times \mathbb{Z}_2$ is abelian but not cyclic.

### Key Properties Summary

```
  Properties of cyclic groups:
  ┌──────────────────────────────────────────────────┐
  │  1. Always abelian                                │
  │  2. Every subgroup is cyclic                      │
  │  3. Unique subgroup of each divisor order         │
  │  4. Classified: ≅ Z  or  ≅ Zₙ                    │
  │  5. Quotients of cyclic groups are cyclic         │
  │  6. If |G| = p (prime), then G ≅ Zₚ              │
  └──────────────────────────────────────────────────┘
```

## 6. Groups of Prime Order

**Theorem 1.21.** If $|G| = p$ where $p$ is prime, then $G$ is cyclic and $G \cong \mathbb{Z}_p$.

*Proof.* Let $a \in G$, $a \neq e$. Then $|\langle a \rangle|$ divides $|G| = p$ (Lagrange). Since $|a| > 1$, we have $|\langle a \rangle| = p$, so $\langle a \rangle = G$. $\square$

**Consequence:** Every element $a \neq e$ in a group of prime order is a generator.

## 7. Classification of Small Cyclic Groups

| Order | Cyclic? | Group | Generators |
|-------|---------|-------|-----------|
| 1 | Yes | $\{e\}$ | $e$ |
| 2 | Yes | $\mathbb{Z}_2$ | $1$ |
| 3 | Yes | $\mathbb{Z}_3$ | $1, 2$ |
| 4 | Maybe | $\mathbb{Z}_4$ (cyclic) or $V_4$ (not) | $1, 3$ for $\mathbb{Z}_4$ |
| 5 | Yes | $\mathbb{Z}_5$ | $1, 2, 3, 4$ |
| 6 | Maybe | $\mathbb{Z}_6$ (cyclic) or $S_3$ (not) | $1, 5$ for $\mathbb{Z}_6$ |

## 8. Roots of Unity — A Key Example

The $n$-th roots of unity form a cyclic group:

$$\mu_n = \{e^{2\pi i k/n} : k = 0, 1, \ldots, n-1\} = \langle e^{2\pi i/n} \rangle$$

under multiplication in $\mathbb{C}^\times$.

**Visualisation for $\mu_6$ (6th roots of unity):**

```
           Im
            ↑
            |    ω = e^(πi/3)
       ω²   |  ω
         \  |  /
          \ | /
   ω³ = −1 ●───────── 1 = ω⁰   → Re
          / | \
         /  |  \
       ω⁴  |  ω⁵
            |
            
   where ω = e^(2πi/6) = cos(60°) + i·sin(60°)
   
   ω⁰ = 1
   ω¹ = (1 + i√3)/2
   ω² = (−1 + i√3)/2
   ω³ = −1
   ω⁴ = (−1 − i√3)/2
   ω⁵ = (1 − i√3)/2
```

$\mu_n \cong \mathbb{Z}_n$, and the **primitive** $n$-th roots of unity (generators) are $e^{2\pi i k/n}$ with $\gcd(k, n) = 1$.

## 9. The Euler Totient Function $\varphi(n)$

Since generators of $\mathbb{Z}_n$ are characterised by $\gcd(a, n) = 1$, the function $\varphi(n) = |\mathbb{Z}_n^\times|$ is fundamental:

$$\varphi(n) = n \prod_{p \mid n} \left(1 - \frac{1}{p}\right)$$

| $n$ | $\varphi(n)$ | Formula |
|-----|-------------|---------|
| 1 | 1 | — |
| 2 | 1 | $2(1 - 1/2) = 1$ |
| 6 | 2 | $6(1-1/2)(1-1/3) = 2$ |
| 8 | 4 | $8(1-1/2) = 4$ |
| 12 | 4 | $12(1-1/2)(1-1/3) = 4$ |
| $p$ | $p - 1$ | $p(1 - 1/p)$ |
| $p^k$ | $p^{k-1}(p-1)$ | $p^k(1-1/p)$ |

**Important identity:** $\sum_{d \mid n} \varphi(d) = n$

## 10. Real-World Applications

| Application | Cyclic Group |
|-------------|-------------|
| Clock arithmetic | $\mathbb{Z}_{12}$ |
| Days of the week | $\mathbb{Z}_7$ |
| RSA cryptography | $\mathbb{Z}_n^\times$ |
| Discrete Fourier Transform | $\mu_n$ (roots of unity) |
| Circular shift registers | $\mathbb{Z}_{2^n - 1}$ |
