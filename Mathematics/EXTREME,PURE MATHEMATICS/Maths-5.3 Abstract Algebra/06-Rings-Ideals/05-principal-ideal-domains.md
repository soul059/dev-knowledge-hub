## Overview

A **principal ideal domain (PID)** is an integral domain where every ideal is principal (generated by a single element). PIDs enjoy beautiful factorisation properties: every PID is a UFD, nonzero primes are maximal, and every finitely generated module over a PID has a nice structure theorem.

---

## 1. Definition

**Definition.** A **principal ideal domain (PID)** is an integral domain $R$ in which every ideal is principal:

$$\forall I \trianglelefteq R,\ \exists a \in R:\ I = (a) = aR$$

```
  PID = Integral domain + every ideal is principal
  
  Chain of ring types:
  
  Field ⊂ Euclidean Domain ⊂ PID ⊂ UFD ⊂ Integral Domain
  
  Every field is a ED is a PID is a UFD is a domain.
  (None of these inclusions reverse.)
```

---

## 2. Key Examples

| Ring | PID? | Reason |
|------|------|--------|
| $\mathbb{Z}$ | ✓ | Every ideal = $n\mathbb{Z} = (n)$ |
| $F[x]$ ($F$ a field) | ✓ | Division algorithm exists |
| $\mathbb{Z}[i]$ | ✓ | Euclidean domain (norm $N(a+bi) = a^2 + b^2$) |
| $\mathbb{Z}[\sqrt{-2}]$ | ✓ | Euclidean domain (norm $N = a^2 + 2b^2$) |
| $\mathbb{Z}[x]$ | ✗ | $(2, x)$ is not principal |
| $\mathbb{Z}[\sqrt{-5}]$ | ✗ | Not even a UFD |
| $F[x, y]$ | ✗ | $(x, y)$ is not principal |
| Any field $F$ | ✓ | Only ideals are $(0)$ and $(1) = F$ |

## 3. Properties of PIDs

### 3.1. Nonzero Primes Are Maximal

**Theorem 6.15.** In a PID, every nonzero prime ideal is maximal.

*Proof.* Let $(p)$ be a nonzero prime ideal in PID $R$, and suppose $(p) \subseteq (a)$. Then $p = ab$ for some $b$. Since $(p)$ is prime and $p = ab \in (p)$: $a \in (p)$ or $b \in (p)$.

- If $a \in (p)$: $(a) \subseteq (p)$, so $(a) = (p)$.
- If $b \in (p)$: $b = pc$, so $p = apc$, giving $1 = ac$ (cancel $p$; domain). So $a$ is a unit, $(a) = R$.

Either way, no ideal strictly between $(p)$ and $R$. $\blacksquare$

### 3.2. Irreducible $\iff$ Prime

**Theorem 6.16.** In a PID, an element is irreducible iff it is prime.

### 3.3. Every PID Is a UFD

**Theorem 6.17.** Every PID is a unique factorisation domain.

*Proof sketch.* 
**Existence:** If $a$ is not irreducible, write $a = bc$ with neither a unit. Repeat. An ascending chain of ideals $(a) \subsetneq (b) \subsetneq \cdots$ must stabilise (ACC in PIDs), so the process terminates.

**Uniqueness:** Irreducible $\implies$ prime (by 6.16), and primes give unique factorisation. $\blacksquare$

### 3.4. GCD Exists

In a PID, for any $a, b \in R$:

$$\gcd(a, b) = d \quad \text{where} \quad (d) = (a) + (b) = (a, b)$$

This gives a **Bézout identity**: $d = ra + sb$ for some $r, s \in R$.

---

## 4. The ACC (Ascending Chain Condition)

**Theorem 6.18.** A PID satisfies the **ascending chain condition (ACC)** on ideals:

Every ascending chain $I_1 \subseteq I_2 \subseteq I_3 \subseteq \cdots$ stabilises (i.e., $\exists N: I_n = I_N$ for all $n \geq N$).

*Proof.* $I = \bigcup I_n$ is an ideal. Since $R$ is a PID, $I = (a)$ for some $a$. Then $a \in I_N$ for some $N$, so $I = (a) \subseteq I_N \subseteq I$. Thus $I_n = I_N$ for all $n \geq N$. $\blacksquare$

Equivalently: a PID is **Noetherian**.

### Ideals of $\mathbb{Z}$

|-------|----------|--------|----------|

| $3\mathbb{Z}$ | 3 | Yes | Yes |

| $6\mathbb{Z}$ | 6 | No | No |
| $p\mathbb{Z}$ | $p$ (prime) | Yes | Yes |

| $(x^2+1)$ | $x^2+1$ | $\mathbb{C}$ | Yes | Yes |

| $(x^2+bx+c)$ irred. | quad. | field ext. | Yes | Yes |

## 6. Modules Over PIDs (Preview)

The FTFGAG generalises to:

**Theorem (Structure of f.g. modules over PIDs).** Every finitely generated module over a PID $R$ decomposes as:

$$M \cong R^r \oplus R/(a_1) \oplus R/(a_2) \oplus \cdots \oplus R/(a_k)$$

where $a_1 \mid a_2 \mid \cdots \mid a_k$ (invariant factors, each non-zero, non-unit).

| Ring | Module | Structure theorem gives |
|------|--------|------------------------|
| $\mathbb{Z}$ | f.g. abelian groups | FTFGAG |
| $F[x]$ | f.g. $F[x]$-modules | Rational/Jordan canonical form |

## 7. When Is $\mathbb{Z}[\sqrt{d}]$ a PID?

For $d < 0$: $\mathbb{Z}[\sqrt{d}]$ is a PID for $d \in \{-1, -2, -3, -7, -11, -19, -43, -67, -163\}$ (the **Heegner numbers** — though technically one considers the ring of integers $\mathcal{O}_K$, which may be larger than $\mathbb{Z}[\sqrt{d}]$).

For $d > 0$: harder; infinitely many, but exact characterisation unknown.

```
  Imaginary quadratic fields K = ℚ(√d), d < 0
  
  The ring of integers O_K is a PID for exactly 9 values of d:
  
  d = -1, -2, -3, -7, -11, -19, -43, -67, -163
  
  These are the Heegner numbers.
  (Proved by Stark and Heegner/Baker.)
```

## 8. PID vs Non-PID: A Comparison

| Property | PID | Non-PID (e.g., $\mathbb{Z}[x]$) |
|----------|-----|----------------------------------|
| Every ideal principal | Yes | No |
| Nonzero prime $\iff$ maximal | Yes | No |
| UFD | Yes | Maybe (Z[x] is UFD) |
| Bézout identity | Always | Not always |
| Noetherian | Yes | Maybe |
| Module structure theorem | Yes | No (in this form) |
